## 1. Επισκόπηση

Το SEPOffice είναι μια σουίτα γραφείου επιτραπέζιου υπολογιστή με τεχνητή νοημοσύνη για Windows που αποτελείται από τρεις ενσωματωμένες εφαρμογές:

| Εφαρμογή | Λειτουργία |
|----------|-----------|
| **SEPWrite** | Δημιουργία εγγράφων εμπλουτισμένου κειμένου, εξαγωγή σε Word/PDF |
| **SEPGrid** | Υπολογιστικό φύλλο με τύπους και γραφήματα |
| **SEPShow** | Παρουσιάσεις με επεξεργαστή καμβά |

Και οι τρεις εφαρμογές συνδέονται με τον **Eliot** – τον ενσωματωμένο βοηθό AI που γράφει κείμενο, δημιουργεί τύπους και σχεδιάζει διαφάνειες.

## 2. Εγκατάσταση και Εκκίνηση

### Πρόγραμμα Εγκατάστασης
Εκτελέστε το `SEPOffice Setup 1.0.1.exe`. Ο οδηγός θα σας καθοδηγήσει στα βήματα εγκατάστασης:
- Μπορεί να επιλεγεί φάκελος εγκατάστασης (προεπιλογή: `Program Files\SEPOffice`)
- Δημιουργία στοιχείου μενού Έναρξη
- Συντόμευση επιφάνειας εργασίας (προαιρετικό)

### Άμεση Εκτέλεση (χωρίς εγκατάσταση)
Εκτελέστε το `release\win-unpacked\SEPOffice.exe`.

### Πρώτη Εκκίνηση
Κατά την πρώτη εκτέλεση, η υπηρεσία AI φορτώνει το γλωσσικό μοντέλο (Qwen2.5-0.5B) στο παρασκήνιο. Η πρόοδος φαίνεται στο γραφικό widget συνομιλίας Eliot κάτω δεξιά. Ανάλογα με το υλικό, μπορεί να χρειαστεί **1–5 λεπτά**.

> **Σημείωση:** Η εφαρμογή είναι άμεσα έτοιμη για χρήση – ο Eliot ενεργοποιείται όταν η μπάρα προόδου φτάσει στο 100%.

## 5. SEPGrid – Υπολογιστικό Φύλλο

Το SEPGrid είναι ένα ισχυρό υπολογιστικό φύλλο με **10.000 γραμμές × 26 στήλες** ανά φύλλο – παρόμοιο με το Microsoft Excel και το OpenOffice Calc.

### 5.1 Βασικές Λειτουργίες

| Ενέργεια | Περιγραφή |
|---------|----------|
| Κλικ κελιού | Επιλογή κελιού |
| Διπλό κλικ / F2 | Επεξεργασία κελιού |
| `Enter` | Επιβεβαίωση, επόμενη γραμμή |
| `Tab` | Επιβεβαίωση, επόμενη στήλη |
| `Escape` | Ακύρωση επεξεργασίας |
| `Ctrl + Z` | Αναίρεση |
| `Ctrl + Y` | Επανάληψη |
| `Ctrl + C` | Αντιγραφή |
| `Ctrl + V` | Επικόλληση |
| `Ctrl + X` | Αποκοπή |
| `Ctrl + B` | Έντονο |
| `Ctrl + I` | Πλάγιο |
| `Ctrl + U` | Υπογράμμιση |
| `Ctrl + F` | Εύρεση και αντικατάσταση |
| `Del` | Εκκαθάριση περιεχομένου κελιού |

**Πολλαπλή επιλογή:** Σύρετε το ποντίκι ή `Shift + κλικ`.

**Λαβή πλήρωσης:** Σύρετε το μπλε τετράγωνο στην κάτω δεξιά γωνία ενός κελιού για αντιγραφή τιμής ή τύπου.

### 5.2 Γραμμή Τύπων

```
┌─────────┬────┬───────────────────────────────────────────┐
│   A1    │ fx │  =SUM(A1:A10)                             │
└─────────┴────┴───────────────────────────────────────────┘
  ↑          ↑              ↑
  Διεύθυνση κελ.  Σύμβολο        Πεδίο εισαγωγής (εμφανίζει ακατέργαστο τύπο)
```

- **Διεύθυνση κελιού**: Εμφανίζει το ενεργό κελί ή επιλογή (π.χ.: `A1:B5`)
- **Σύμβολο fx**: Υποδεικνύει εισαγωγή τύπου
- **Πεδίο εισαγωγής**: Εμφανίζει πάντα τον ακατέργαστο τύπο – όχι το αποτέλεσμα

**Εμφάνιση σφάλματος:** Για σφάλματα τύπου όπως `#ΣΦΑΛΜΑ`, `#REF!`, `#DIV/0!` το πεδίο περιβάλλεται από κόκκινο περίγραμμα.

### 5.3 Εισαγωγή Τύπων

Η εισαγωγή τύπου πάντα ξεκινά με `=`:

```
=SUM(A1:A10)               Άθροισμα
=AVERAGE(B1:B5)            Μέσος όρος
=MAX(C1:C100)              Μέγιστη τιμή
=MIN(D1:D50)               Ελάχιστη τιμή
=COUNT(A:A)                Καταμέτρηση
=IF(A1>0;"Ναι";"Όχι")      Συνθήκη
=VLOOKUP(...)              Κατακόρυφη αναζήτηση
=ROUND(A1; 2)              Στρογγυλοποίηση
```

#### Αναφορά τύπου με κλικ ποντικιού

1. Πληκτρολογήστε `=` ή συνάρτηση όπως `=SUM(`
2. Κάντε κλικ στο επιθυμητό κελί → εισάγεται η διεύθυνση
3. Σύρετε για εύρος → εισάγεται το εύρος (π.χ.: `A1:B10`)

### 5.4 Τύποι Μεταξύ Φύλλων

Αναφορά σε άλλα φύλλα – ακριβώς όπως στο Excel:

```
=Sheet2!A1                    Μεμονωμένο κελί από Sheet2
=SUM(Sheet2!A1:B10)           Άθροισμα από Sheet2
=Sheet1!A1 + Sheet2!B5        Συνδυασμός πολλαπλών φύλλων
=AVERAGE(Πωλήσεις!C1:C100)    Εύρος από φύλλο 'Πωλήσεις'
```

**Με κλικ ποντικιού:**
1. Πληκτρολογήστε `=` σε κελί
2. Κάντε κλικ σε άλλη καρτέλα φύλλου → εισάγεται `Sheet2!`
3. Κάντε κλικ στο επιθυμητό κελί → προστίθεται `A1`

### 5.5 Μορφή Αριθμών

Από το **σύμβολο 123** στη γραμμή εργαλείων:

| Μορφή | Παράδειγμα | Περιγραφή |
|-------|-----------|----------|
| Γενική | 1234.5 | Χωρίς μορφοποίηση |
| Αριθμός | 1.234,56 | Μορφή 2 δεκαδικών ψηφίων |
| Νόμισμα | 1.234,56 € | Μορφή νομίσματος |
| Ποσοστό | 12,5% | Τιμή×100 με σύμβολο % |
| Χιλιάδες | 1.234 | Ακέραιος με διαχωριστή χιλιάδων |
| Ημερομηνία | 24/02/2026 | Μορφή ημερομηνίας |

### 5.6 Συγχώνευση Κελιών

1. Επιλέξτε εύρος κελιών
2. **Εισαγωγή → Συγχώνευση κελιών**
3. Διατηρείται μόνο η τιμή του επάνω αριστερού κελιού

**Κατάργηση συγχώνευσης:** **Εισαγωγή → Κατάργηση συγχώνευσης κελιών**

### 5.7 Μορφοποίηση Υπό Συνθήκη

1. Επιλέξτε εύρος
2. **Επεξεργασία → Μορφοποίηση υπό συνθήκη...**
3. Επιλέξτε κανόνα: Μεγαλύτερο από / Μικρότερο από / Ίσο με / Μεταξύ / Περιέχει κείμενο
4. Επιλέξτε χρώμα κειμένου και φόντου
5. **Προσθήκη**

**Διαγραφή:** Επιλέξτε εύρος → **Επεξεργασία → Διαγραφή μορφοποίησης υπό συνθήκη**

### 5.8 Πάγωμα Γραμμών/Στηλών

1. Επιλέξτε το κελί από το οποίο θα ξεκινά η κύλιση
2. **Προβολή → Πάγωμα γραμμών/στηλών**
3. Παγώνουν όλες οι γραμμές από πάνω και οι στήλες αριστερά

**Κατάργηση παγώματος:** **Προβολή → Κατάργηση παγώματος**

### 5.9 Σχόλια

1. Επιλέξτε κελί → **Εισαγωγή → Προσθήκη σχολίου** → Εισαγωγή κειμένου → OK
2. Τα κελιά με σχόλια εμφανίζουν **κόκκινο τρίγωνο** στην επάνω δεξιά γωνία

### 5.10 Εύρεση και Αντικατάσταση

`Ctrl + F` ή **Επεξεργασία → Εύρεση και αντικατάσταση**:
- **Εύρεση**: Πλοήγηση με ◀ ▶
- **Αντικατάσταση**: Ένα-ένα ή όλα μαζί

### 5.11 Εισαγωγή/Διαγραφή Γραμμών και Στηλών

Από το μενού **Εισαγωγή**: Εισαγωγή γραμμής πάνω/κάτω, εισαγωγή στήλης αριστερά/δεξιά, διαγραφή γραμμής/στήλης.

### 5.12 Γραμμή Κατάστασης

Στο κάτω μέρος, τα στατιστικά πολλαπλής επιλογής εμφανίζονται αυτόματα:

```
Σ Άθρ: 12.345   ⌀ ΜΟ: 1.234   ↓ Ελάχ: 100   ↑ Μέγ: 5.000   Πλήθος: 10
```

### 5.13 Εισαγωγή / Εξαγωγή

| Λειτουργία | Μορφή | Περιγραφή |
|-----------|-------|----------|
| Εισαγωγή | `.xlsx`, `.xlsm` | Άνοιγμα αρχείου Excel |
| Εισαγωγή CSV | `.csv` | Αρχείο με διαχωρισμό κόμματος/ερωτηματικού |
| Εξαγωγή | `.xlsx` | Αποθήκευση ως αρχείο Excel |
| Εξαγωγή CSV | `.csv` | Αποθήκευση ως CSV (UTF-8, ερωτηματικό) |

| Εισαγωγή εικόνας | PNG, JPG | Ενσωμάτωση εικόνας στο υπολογιστικό φύλλο |

### 5.14 Λειτουργίες AI

## 6. SEPShow – Παρουσίαση

Το SEPShow είναι ένας επεξεργαστής παρουσίασης διαφανειών που βασίζεται στο **Konva** (μηχανισμός απόδοσης καμβά).

### 6.1 Διεπαφή

| Περιοχή | Περιγραφή |
|--------|----------|
| **Αριστερή πλαϊνή μπάρα** | Λίστα διαφανειών – προεπισκόπηση, σύρετε για αλλαγή σειράς |
| **Κεντρική περιοχή** | Επεξεργαστής καμβά για την ενεργή διαφάνεια |
| **Γραμμή εργαλείων** | Εργαλεία, εξαγωγή, λειτουργία παρουσίασης |
| **Σημειώσεις παρουσιαστή** | Σημειώσεις για την τρέχουσα διαφάνεια |

### 6.2 Λειτουργία Παρουσίασης

Κάντε κλικ στο **Παρουσίαση** για παρουσίαση πλήρους οθόνης:
- Πλοήγηση: Πλήκτρα βέλους `→` / `←` ή κλικ
- `Esc` για έξοδο από τη λειτουργία παρουσίασης
