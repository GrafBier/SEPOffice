## 1. अवलोकन

SEPOffice Windows के लिए एक AI-संचालित डेस्कटॉप ऑफिस सूट है जिसमें तीन एकीकृत अनुप्रयोग हैं:

| ऐप | कार्य |
|----|-------|
| **SEPWrite** | रिच टेक्स्ट दस्तावेज़ बनाना, Word/PDF में निर्यात |
| **SEPGrid** | सूत्रों और चार्ट के साथ स्प्रेडशीट |
| **SEPShow** | कैनवास एडिटर के साथ प्रेजेंटेशन |

तीनों ऐप **Eliot** से जुड़े हैं – एक अंतर्निहित AI सहायक जो टेक्स्ट लिखता है, सूत्र उत्पन्न करता है और स्लाइड डिज़ाइन करता है।

## 2. इंस्टॉलेशन और प्रारंभ

### इंस्टॉलर
`SEPOffice Setup 1.0.1.exe` चलाएं। विज़ार्ड इंस्टॉलेशन चरणों के माध्यम से मार्गदर्शन करेगा:
- इंस्टॉलेशन फ़ोल्डर चुन सकते हैं (डिफ़ॉल्ट: `Program Files\SEPOffice`)
- स्टार्ट मेनू आइटम बनाएं
- डेस्कटॉप शॉर्टकट (वैकल्पिक)

### सीधे चलाएं (इंस्टॉलेशन के बिना)
`release\win-unpacked\SEPOffice.exe` चलाएं।

### पहला प्रारंभ
पहली बार चलाने पर, AI सेवा पृष्ठभूमि में भाषा मॉडल (Qwen2.5-0.5B) लोड करती है। प्रगति दाईं ओर नीचे Eliot चैट विजेट में दिखाई देती है। हार्डवेयर के आधार पर **1–5 मिनट** लग सकते हैं।

> **नोट:** ऐप तुरंत उपयोग के लिए तैयार है – Eliot प्रगति बार 100% होने पर सक्रिय होता है।

## 5. SEPGrid – स्प्रेडशीट

SEPGrid एक शक्तिशाली स्प्रेडशीट है जिसमें प्रति शीट **10,000 पंक्तियाँ × 26 कॉलम** हैं – Microsoft Excel और OpenOffice Calc के समान।

### 5.1 बुनियादी ऑपरेशन

| क्रिया | विवरण |
|--------|-------|
| सेल क्लिक | सेल चुनें |
| डबल-क्लिक / F2 | सेल संपादित करें |
| `Enter` | पुष्टि करें, अगली पंक्ति |
| `Tab` | पुष्टि करें, अगला कॉलम |
| `Escape` | संपादन रद्द करें |
| `Ctrl + Z` | पूर्ववत करें |
| `Ctrl + Y` | फिर से करें |
| `Ctrl + C` | कॉपी |
| `Ctrl + V` | पेस्ट |
| `Ctrl + X` | काटें |
| `Ctrl + B` | बोल्ड |
| `Ctrl + I` | इटैलिक |
| `Ctrl + U` | रेखांकित |
| `Ctrl + F` | ढूंढें और बदलें |
| `Del` | सेल सामग्री साफ़ करें |

**बहु-चयन:** माउस ड्रैग करें या `Shift + क्लिक`।

**फिल हैंडल:** मान या सूत्र कॉपी करने के लिए सेल के निचले दाएं कोने में नीले वर्ग को ड्रैग करें।

### 5.2 सूत्र बार

```
┌─────────┬────┬───────────────────────────────────────────┐
│   A1    │ fx │  =SUM(A1:A10)                             │
└─────────┴────┴───────────────────────────────────────────┘
  ↑          ↑              ↑
  सेल पता      प्रतीक          इनपुट फ़ील्ड (कच्चा सूत्र दिखाता है)
```

- **सेल पता**: सक्रिय सेल या चयन दिखाता है (उदाहरण: `A1:B5`)
- **प्रतीक fx**: सूत्र इनपुट दर्शाता है
- **इनपुट फ़ील्ड**: हमेशा कच्चा सूत्र दिखाता है – परिणाम नहीं

**त्रुटि प्रदर्शन:** `#त्रुटि`, `#REF!`, `#DIV/0!` जैसी सूत्र त्रुटियों पर फ़ील्ड के चारों ओर लाल बॉर्डर दिखाई देता है।

### 5.3 सूत्र दर्ज करना

सूत्र इनपुट हमेशा `=` से शुरू होता है:

```
=SUM(A1:A10)                  योग
=AVERAGE(B1:B5)               औसत
=MAX(C1:C100)                 अधिकतम
=MIN(D1:D50)                  न्यूनतम
=COUNT(A:A)                   गिनती
=IF(A1>0;"हाँ";"नहीं")        शर्त
=VLOOKUP(...)                 वर्टिकल लुकअप
=ROUND(A1; 2)                 गोलाई
```

#### माउस क्लिक से सूत्र संदर्भ

1. `=` या `=SUM(` जैसा फ़ंक्शन टाइप करें
2. इच्छित सेल क्लिक करें → पता सम्मिलित होता है
3. रेंज के लिए ड्रैग करें → रेंज सम्मिलित होती है (उदाहरण: `A1:B10`)

### 5.4 शीट के पार सूत्र

अन्य शीट का संदर्भ – Excel के समान:

```
=Sheet2!A1                    Sheet2 का एकल सेल
=SUM(Sheet2!A1:B10)           Sheet2 का योग
=Sheet1!A1 + Sheet2!B5        कई शीट का संयोजन
=AVERAGE(बिक्री!C1:C100)      'बिक्री' शीट की रेंज
```

**माउस क्लिक से:**
1. सेल में `=` टाइप करें
2. किसी अन्य शीट टैब पर क्लिक करें → `Sheet2!` सम्मिलित होता है
3. इच्छित सेल क्लिक करें → `A1` जोड़ा जाता है

### 5.5 संख्या प्रारूप

टूलबार पर **123 प्रतीक** से:

| प्रारूप | उदाहरण | विवरण |
|---------|--------|-------|
| सामान्य | 1234.5 | कोई प्रारूप नहीं |
| संख्या | 1,234.56 | 2 दशमलव स्थान प्रारूप |
| मुद्रा | ₹1,234.56 | मुद्रा प्रारूप |
| प्रतिशत | 12.5% | मान×100 पर % चिह्न |
| हजार | 1,234 | हजार विभाजक के साथ पूर्णांक |
| दिनांक | 24/02/2026 | दिनांक प्रारूप |

### 5.6 सेल मर्ज करना

1. सेल रेंज चुनें
2. **सम्मिलित > सेल मर्ज करें**
3. केवल ऊपरी-बाईं सेल का मान बना रहता है

**मर्ज हटाएं:** **सम्मिलित > सेल मर्ज हटाएं**

### 5.7 सशर्त स्वरूपण

1. रेंज चुनें
2. **संपादित > सशर्त स्वरूपण...**
3. नियम चुनें: से बड़ा / से छोटा / बराबर / के बीच / टेक्स्ट शामिल है
4. टेक्स्ट और पृष्ठभूमि रंग चुनें
5. **जोड़ें**

**हटाएं:** रेंज चुनें → **संपादित > सशर्त स्वरूपण हटाएं**

### 5.8 पंक्ति/कॉलम स्थिर करना

1. जिस सेल से स्क्रॉलिंग शुरू होनी चाहिए उसे चुनें
2. **देखें > पंक्ति/कॉलम स्थिर करें**
3. ऊपर की सभी पंक्तियां और बाईं ओर के कॉलम स्थिर हो जाते हैं

**स्थिर हटाएं:** **देखें > पंक्ति/कॉलम स्थिर हटाएं**

### 5.9 टिप्पणियां

1. सेल चुनें → **सम्मिलित > टिप्पणी जोड़ें** → टेक्स्ट दर्ज करें → ठीक
2. टिप्पणी वाले सेल के ऊपरी दाएं कोने में **लाल त्रिभुज** दिखाई देता है

### 5.10 ढूंढें और बदलें

`Ctrl + F` या **संपादित > ढूंढें और बदलें**:
- **ढूंढें**: ◀ ▶ से नेविगेट करें
- **बदलें**: एक-एक करके या सभी एक साथ

### 5.11 पंक्तियां और कॉलम सम्मिलित करना/हटाना

**सम्मिलित** मेनू से: ऊपर/नीचे पंक्ति सम्मिलित करें, बाईं/दाईं ओर कॉलम सम्मिलित करें, पंक्ति/कॉलम हटाएं।

### 5.12 स्थिति बार

नीचे बहु-चयन आंकड़े स्वचालित रूप से दिखाई देते हैं:

```
Σ योग: 12,345   ⌀ औसत: 1,234   ↓ न्यूनतम: 100   ↑ अधिकतम: 5,000   गिनती: 10
```

### 5.13 आयात / निर्यात

| कार्य | प्रारूप | विवरण |
|-------|---------|-------|
| आयात | `.xlsx`, `.xlsm` | Excel फ़ाइल खोलें |
| CSV आयात | `.csv` | कॉमा/सेमीकोलन से अलग फ़ाइल |
| निर्यात | `.xlsx` | Excel फ़ाइल के रूप में सहेजें |
| CSV निर्यात | `.csv` | CSV के रूप में सहेजें (UTF-8, सेमीकोलन) |
| प्रिंट / PDF | — | प्रिंट प्रीव्यू, केवल भरी हुई पंक्तियां |
| छवि सम्मिलित करें | PNG, JPG | स्प्रेडशीट में छवि एम्बेड करें |

## 6. SEPShow – प्रेजेंटेशन

| क्षेत्र | विवरण |
