## 1. نظرة عامة

SEPOffice هو حزمة مكتبية لسطح المكتب مدعومة بالذكاء الاصطناعي لنظام Windows، وتتكون من ثلاثة تطبيقات متكاملة:

| التطبيق | الوظيفة |
|---------|---------|
| **SEPWrite** | إنشاء مستندات نصية منسقة، التصدير إلى Word/PDF |
| **SEPGrid** | جداول بيانات مع صيغ ورسوم بيانية |
| **SEPShow** | عروض تقديمية مع محرر على القماش |

تتصل جميع التطبيقات الثلاثة بـ **Eliot** – مساعد ذكاء اصطناعي مدمج يصيغ النصوص ويولد الصيغ ويصمم الشرائح.

---

## 2. التثبيت والتشغيل

### برنامج التثبيت
قم بتشغيل `SEPOffice Setup 1.0.1.exe`. سيرشدك المعالج خلال عملية التثبيت:
- مجلد تثبيت قابل للتحديد (الافتراضي: `Program Files\SEPOffice`)
- إنشاء عنصر في قائمة البدء
- اختصار اختياري على سطح المكتب

### التشغيل المباشر (بدون تثبيت)
قم بتشغيل `release\win-unpacked\SEPOffice.exe`.

### أول تشغيل
عند أول تشغيل، تحمّل خدمة الذكاء الاصطناعي النموذج اللغوي (Qwen2.5-0.5B) في الخلفية. يظهر التقدم في أداة دردشة Eliot في أسفل اليمين. اعتمادًا على الأجهزة، يستغرق هذا **1–5 دقائق**.

> **ملاحظة:** التطبيق قابل للاستخدام فورًا – يصبح Eliot نشطًا عندما يصل شريط التقدم إلى 100%.

## 5. SEPGrid – جداول البيانات

SEPGrid هو جدول بيانات قوي بـ **10,000 صف × 26 عمود** لكل ورقة – مشابه لـ Microsoft Excel وOpenOffice Calc.

### 5.1 الاستخدام الأساسي

| الإجراء | الوصف |
|---------|-------|
| النقر على خلية | تحديد خلية |
| النقر المزدوج / F2 | تحرير خلية |
| `Enter` | تأكيد، الانتقال للصف التالي |
| `Tab` | تأكيد، الانتقال للعمود التالي |
| `Escape` | إلغاء التحرير |
| `Ctrl + Z` | تراجع |
| `Ctrl + Y` | إعادة |
| `Ctrl + C` | نسخ |
| `Ctrl + V` | لصق |
| `Ctrl + X` | قص |
| `Ctrl + B` | غامق |
| `Ctrl + I` | مائل |
| `Ctrl + U` | مسطّر |
| `Ctrl + F` | بحث واستبدال |
| `Del` | مسح محتوى الخلية |

**تحديد متعدد:** اسحب الفأرة أو `Shift + نقر`.

**مقبض التعبئة:** اسحب المربع الأزرق في أسفل يمين الخلية لنسخ القيم أو الصيغ.

### 5.2 شريط الصيغ

```
┌─────────┬────┬───────────────────────────────────────────┐
│   A1    │ fx │  =SUM(A1:A10)                             │
└─────────┴────┴───────────────────────────────────────────┘
  ↑          ↑              ↑
  عنوان الخل.  رمز            حقل الإدخال (يعرض الصيغة الخام)
```

- **عنوان الخلية**: يعرض الخلية النشطة أو النطاق المحدد (مثلاً `A1:B5`)
- **رمز fx**: يشير إلى إدخال صيغة
- **حقل الإدخال**: يعرض دائمًا الصيغة الخام – لا النتيجة

**عرض الأخطاء:** عند أخطاء الصيغة مثل `#خطأ`، `#مرجع!`، `#قسمة/0!` إلخ، يُحاط الحقل بإطار أحمر وتظهر الرسالة على اليمين.

### 5.3 إدخال الصيغ

إدخال الصيغة يبدأ دائمًا بـ `=`:

```
=SUM(A1:A10)               مجموع
=AVERAGE(B1:B5)            متوسط
=MAX(C1:C100)              أقصى قيمة
=MIN(D1:D50)               أدنى قيمة
=COUNT(A:A)                عدد
=IF(A1>0;"نعم";"لا")       شرط
=VLOOKUP(...)              بحث رأسي
=ROUND(A1; 2)              تقريب
```

#### مراجع الصيغ بنقر الفأرة

1. اكتب `=` أو دالة مثل `=SUM(`
2. انقر على الخلية المطلوبة ← سيُدرج العنوان
3. اسحب للنطاقات ← سيُدرج النطاق (مثلاً `A1:B10`)

### 5.4 الصيغ بين الأوراق

المراجع إلى أوراق أخرى – كما في Excel:

```
=Sheet2!A1                    خلية منفردة من Sheet2
=SUM(Sheet2!A1:B10)           مجموع من Sheet2
=Sheet1!A1 + Sheet2!B5        مجموعة من أوراق متعددة
=AVERAGE(Sales!C1:C100)       نطاق من ورقة "Sales"
```

**عبر نقر الفأرة:**
1. اكتب `=` في خلية
2. انقر على تبويب ورقة أخرى ← يُدرج `Sheet2!`
3. انقر على الخلية المطلوبة ← يُضيف `A1`

### 5.5 تنسيقات الأرقام

عبر رمز **123** في شريط الأدوات:

| التنسيق | مثال | الوصف |
|---------|-----|-------|
| قياسي | 1234.5 | بلا تنسيق |
| رقم | 1,234.56 | تنسيق بخانتين عشريتين |
| عملة | 1,234.56 ر.س | تنسيق العملة |
| نسبة مئوية | 12.5% | القيمة × 100 مع علامة % |
| آلاف | 1,234 | أعداد صحيحة مع فاصل الآلاف |
| تاريخ | 24/02/2026 | تنسيق التاريخ |

### 5.6 دمج الخلايا

1. حدد نطاق الخلايا
2. **إدراج ← دمج الخلايا**
3. يُحتفظ فقط بقيمة الخلية العلوية اليسرى

**فك الدمج:** **إدراج ← فك دمج الخلايا**

### 5.7 التنسيق الشرطي

1. حدد النطاق
2. **تحرير ← التنسيق الشرطي...**
3. اختر القاعدة: أكبر من / أصغر من / يساوي / بين / النص يحتوي
4. اختر ألوان النص والخلفية
5. **إضافة**

**الحذف:** حدد النطاق ← **تحرير ← حذف التنسيق الشرطي**

### 5.8 تجميد الأجزاء

1. حدد الخلية التي ستبدأ منها عملية التحريك
2. **عرض ← تجميد الأجزاء**
3. تُجمَّد جميع الصفوف أعلاه والأعمدة على اليسار

**إلغاء التجميد:** **عرض ← إلغاء تجميد الأجزاء**

### 5.9 التعليقات

1. حدد الخلية ← **إدراج ← إضافة تعليق** ← أدخل النص ← موافق
2. الخلايا التي تحتوي على تعليقات تعرض **مثلثًا أحمر** في الركن العلوي الأيمن

### 5.10 البحث والاستبدال

`Ctrl + F` أو **تحرير ← بحث واستبدال**:
- **بحث**: التنقل ◀ ▶
- **استبدال**: فردياً أو الكل دفعة واحدة

### 5.11 إدراج/حذف الصفوف والأعمدة

عبر قائمة **إدراج**: إدراج صف أعلى/أسفل، عمود يسار/يمين، حذف صف/عمود.

### 5.12 شريط الحالة

في الأسفل تظهر إحصاءات تلقائية للتحديدات المتعددة:

```
Σ مجموع: 12,345   ⌀ متوسط: 1,234   ↓ أدنى: 100   ↑ أقصى: 5,000   عدد: 10
```

### 5.13 الاستيراد / التصدير

| الوظيفة | الصيغة | الوصف |
|---------|--------|-------|
| استيراد | `.xlsx`, `.xlsm` | فتح ملفات Excel |
| استيراد CSV | `.csv` | ملفات مفصولة بفاصلة/فاصلة منقوطة |
| تصدير | `.xlsx` | الحفظ كملف Excel |
| تصدير CSV | `.csv` | الحفظ كـ CSV (UTF-8، فاصلة منقوطة) |
| طباعة / PDF | — | معاينة الطباعة، الصفوف الممتلئة فقط |
| إدراج صورة | PNG, JPG | تضمين صورة في الجدول |

### 5.14 وظائف الذكاء الاصطناعي

## 6. SEPShow – العروض التقديمية

SEPShow هو محرر عروض تقديمية مبني على **Konva** (محرك رسم القماش).

### 6.1 الواجهة

| المنطقة | الوصف |
|---------|-------|
| **الشريط الجانبي الأيمن** | قائمة الشرائح – معاينة، تغيير الترتيب بالسحب |
| **المنطقة الرئيسية** | محرر القماش للشريحة النشطة |
| **شريط الأدوات** | الأدوات، التصدير، وضع العرض |
| **ملاحظات المحاضر** | ملاحظات للشريحة الحالية |

### 6.2 وضع العرض التقديمي

انقر على **عرض** للعرض بملء الشاشة:
- التنقل: مفاتيح الأسهم `→` / `←` أو النقر
- `Esc` يُنهي وضع العرض

## 7. Eliot – المساعد الذكي

Eliot هو المساعد الذكاء الاصطناعي المدمج المتاح في التطبيقات الثلاثة.

### 7.1 أداة الدردشة

**رمز Eliot العائم** موجود في أسفل يمين جميع التطبيقات.

#### مؤشرات الحالة

| الرمز | الحالة |
|-------|-------|
| 💬 (طبيعي) | الذكاء الاصطناعي جاهز |
| ⏳ (دوار) | جاري تحميل الذكاء الاصطناعي |
| ❌ (أحمر) | خطأ أو لا يوجد اتصال |

## ملاحظات تقنية

- **تخزين البيانات:** تُخزَّن جميع المستندات محليًا في تخزين المتصفح. لا تُرسَل أي بيانات إلى خوادم خارجية.

- **متطلبات النظام:** Windows 10/11، ذاكرة RAM لا تقل عن 4 GB (يُوصى بـ 8 GB لنموذج الذكاء الاصطناعي)
