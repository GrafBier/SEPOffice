## 1. Обзор

SEPOffice — настольный офисный пакет с ИИ для Windows, состоящий из трёх интегрированных приложений:

| Приложение | Функция |
|-----------|---------|
| **SEPWrite** | Создание документов rich-text, экспорт в Word/PDF |
| **SEPGrid** | Электронные таблицы с формулами и диаграммами |
| **SEPShow** | Презентации с редактором canvas |

Все три приложения подключены к **Eliot** — встроенному ИИ-помощнику, который формулирует тексты, генерирует формулы и создаёт слайды.

## 2. Установка и запуск

### Установщик
Запустите `SEPOffice Setup 1.0.1.exe`. Мастер установки проведёт вас через процесс:
- Выбираемый каталог установки (по умолчанию: `Program Files\SEPOffice`)
- Создаётся запись в меню «Пуск»
- Необязательный ярлык на рабочем столе

### Прямой запуск (без установщика)
Запустите `release\win-unpacked\SEPOffice.exe`.

### Первый запуск
При первом запуске служба ИИ загружает языковую модель (Qwen2.5-0.5B) в фоновом режиме. Прогресс загрузки отображается в виджете чата Eliot в правом нижнем углу. В зависимости от оборудования это занимает **1–5 минут**.

> **Примечание:** Приложение сразу готово к использованию — Eliot активируется, когда полоса загрузки достигает 100%.

## 3. Панель управления

Панель управления — стартовая страница SEPOffice.

### Три плитки приложений
- **SEPWrite** – Создать или открыть документ
- **SEPGrid** – Создать или открыть таблицу

### Последние документы
Под плитками отображаются недавно отредактированные документы с иконкой типа, названием и датой последнего изменения.

| ⌨️ | Показать горячие клавиши |

## 5. SEPGrid – Электронные таблицы

SEPGrid — мощная электронная таблица с **10 000 строк × 26 столбцов** на лист — сопоставимая с Microsoft Excel и OpenOffice Calc.

### 5.1 Основные операции

| Действие | Описание |
|----------|----------|
| Клик по ячейке | Выбрать ячейку |
| Двойной клик / F2 | Редактировать ячейку |
| `Enter` | Подтвердить, перейти к следующей строке |
| `Tab` | Подтвердить, перейти к следующему столбцу |
| `Escape` | Отменить редактирование |
| `Ctrl + Z` | Отменить |
| `Ctrl + Y` | Повторить |
| `Ctrl + C` | Копировать |
| `Ctrl + V` | Вставить |
| `Ctrl + X` | Вырезать |
| `Ctrl + B` | Жирный |
| `Ctrl + I` | Курсив |
| `Ctrl + U` | Подчёркнутый |
| `Ctrl + F` | Найти и Заменить |
| `Del` | Очистить содержимое ячейки |

**Множественный выбор:** Перетащите мышь с нажатой левой кнопкой или `Shift + Клик`.

**Маркер заполнения:** Перетащите синий квадрат в правом нижнем углу ячейки для копирования значений или формул.

### 5.2 Строка формул

```
┌─────────┬────┬───────────────────────────────────────────┐
│   A1    │ fx │  =СУММ(A1:A10)                            │
└─────────┴────┴───────────────────────────────────────────┘
  ↑          ↑              ↑
  Адрес      Символ         Поле ввода (показывает формулу)
```

- **Адрес ячейки**: Показывает активную ячейку или выделенный диапазон (например, `A1:B5`)
- **Символ fx**: Указывает на ввод формулы
- **Поле ввода**: Всегда показывает саму формулу — не результат

**Отображение ошибок:** При ошибках `#ОШИБКА`, `#ССЫЛ!`, `#ДЕЛ/0!` и т.д. поле становится красным и ошибка отображается справа.

### 5.3 Ввод формул

Ввод формулы всегда начинается с `=`:

```
=СУММ(A1:A10)              Сумма
=СРЗНАЧ(B1:B5)             Среднее значение
=МАКС(C1:C100)             Максимум
=МИН(D1:D50)               Минимум
=СЧЁТ(A:A)                 Количество
=ЕСЛИ(A1>0,"Да","Нет")     Условие
=ВПР(...)                  Вертикальный поиск
=ОКРУГЛ(A1; 2)             Округление
```

#### Ссылки на ячейки кликом мыши

1. Введите `=` или функцию, например `=СУММ(`
2. Нажмите на нужную ячейку → Адрес вставляется
3. Перетащите для диапазонов → Диапазон вставляется (например, `A1:B10`)

### 5.4 Формулы между листами

Ссылки на другие листы — как в Excel:

```
=Лист2!A1                        Отдельная ячейка из Лист2
=СУММ(Лист2!A1:B10)              Сумма из Лист2
=Лист1!A1 + Лист2!B5             Комбинация нескольких листов
=СРЗНАЧ(Продажи!C1:C100)         Диапазон из листа "Продажи"
```

**Кликом мыши:**
1. Введите `=` в ячейку
2. Нажмите на другую вкладку листа → `Лист2!` вставляется
3. Нажмите на нужную ячейку → `A1` добавляется

### 5.5 Форматы чисел

Через символ **123** на панели инструментов:

| Формат | Пример | Описание |
|--------|--------|----------|
| Стандартный | 1234.5 | Без форматирования |
| Число | 1 234,56 | Стандартная нотация с 2 десятичными |
| Валюта | 1 234,56 ₽ | Формат валюты |
| Процент | 12,5% | Значение × 100 со знаком % |
| Тысячи | 1 234 | Целые числа с разделением тысяч |
| Дата | 24.02.2026 | Формат даты |

### 5.6 Объединение ячеек

1. Выберите диапазон ячеек
2. **Вставка → Объединить ячейки**
3. Сохраняется только значение верхней левой ячейки

**Разъединить:** **Вставка → Разъединить ячейки**

### 5.7 Условное форматирование

1. Выберите диапазон
2. **Правка → Условное форматирование...**
3. Выберите правило: Больше / Меньше / Равно / Между / Текст содержит
4. Выберите цвета текста и фона
5. **Добавить**

**Удалить:** Выбрать диапазон → **Правка → Удалить условное форматирование**

### 5.8 Закрепить области

1. Выберите ячейку, с которой начинается прокрутка
2. **Вид → Закрепить области**
3. Все строки выше и столбцы слева будут закреплены

**Снять закрепление:** **Вид → Снять закрепление**

### 5.9 Комментарии

1. Выберите ячейку → **Вставка → Добавить комментарий** → Введите текст → ОК
2. Ячейки с комментариями показывают **красный треугольник** в верхнем правом углу

### 5.10 Найти и Заменить

`Ctrl + F` или **Правка → Найти и Заменить**:
- **Найти**: Навигация с ◀ ▶
- **Заменить**: Одиночная замена или все сразу

### 5.11 Вставка/Удаление строк и столбцов

Через меню **Вставка**: Вставить строку выше/ниже, столбец слева/справа, удалить строку/столбец.

### 5.12 Строка состояния

Внизу автоматически отображаются статистики для множественного выбора:

```
Σ Сумма: 12 345   ⌀ Среднее: 1 234   ↓ Мин: 100   ↑ Макс: 5 000   Количество: 10
```

### 5.13 Импорт / Экспорт

| Функция | Формат | Описание |
|---------|--------|----------|
| Импорт | `.xlsx`, `.xlsm` | Открыть файлы Excel |
| Импорт CSV | `.csv` | Файлы с разделителями запятая/точка с запятой |
| Экспорт | `.xlsx` | Сохранить как файл Excel |

| Вставить изображение | PNG, JPG | Встроить изображение в таблицу |

## 6. SEPShow – Презентации

SEPShow — редактор презентаций на основе слайдов с **Konva** (canvas-рендерер).

### 6.1 Интерфейс

| Область | Описание |
|---------|----------|
| **Левая боковая панель** | Список слайдов – предварительный просмотр, изменение порядка перетаскиванием |
| **Главная область** | Canvas-редактор активного слайда |
| **Панель инструментов** | Инструменты, экспорт, режим презентации |
| **Заметки докладчика** | Заметки для текущего слайда |

### 6.2 Вставка элементов

| Действие | Описание |
|----------|----------|
| **Текст** | Разместить текстовое поле на слайде |
| **Изображение** | Вставить изображение через загрузку |
| **Фигура** | Прямоугольники, круги и т.д. |
| **ИИ-макет** | ИИ генерирует полный макет слайда |

### 6.3 Режим презентации

Нажмите **Показать** для полноэкранной презентации:
- Навигация: клавиши `→` / `←` или клик
- `Esc` выходит из режима презентации

## 7. Eliot – ИИ-помощник

Eliot — встроенный ИИ-помощник, доступный во всех трёх приложениях.

### 7.1 Виджет чата

**Плавающий символ Eliot** находится в правом нижнем углу во всех приложениях.

#### Индикаторы состояния

| Символ | Состояние |
|--------|-----------|
| 💬 (обычный) | ИИ готов |
| ⏳ (вращается) | ИИ загружается |
| ❌ (красный) | Ошибка или не подключён |

## Технические примечания

- **Хранение данных:** Все документы сохраняются локально в хранилище браузера. Данные не передаются на внешние серверы.
- **Модель ИИ:** Qwen2.5-0.5B работает полностью локально — интернет-соединение для функций ИИ не требуется.
- **Системные требования:** Windows 10/11, мин. 4 ГБ ОЗУ (8 ГБ рекомендуется для модели ИИ)
