## 1. 개요

SEPOffice는 Windows용 AI 기반 데스크톱 오피스 제품군으로 세 가지 통합 애플리케이션으로 구성됩니다:

| 앱 | 기능 |
|----|------|
| **SEPWrite** | 리치 텍스트 문서 작성, Word/PDF 내보내기 |
| **SEPGrid** | 수식 및 차트가 포함된 스프레드시트 |
| **SEPShow** | 캔버스 편집기를 활용한 프레젠테이션 |

세 앱 모두 **Eliot**에 연결됩니다 – 텍스트를 작성하고, 수식을 생성하며, 슬라이드를 디자인하는 내장 AI 어시스턴트입니다.

## 2. 설치 및 시작

### 설치 프로그램
`SEPOffice Setup 1.0.1.exe`를 실행합니다. 마법사가 설치 단계를 안내합니다:
- 설치 폴더 선택 가능 (기본값: `Program Files\SEPOffice`)
- 시작 메뉴 항목 생성
- 바탕화면 바로가기 (선택 사항)

### 직접 실행 (설치 없이)
`release\win-unpacked\SEPOffice.exe`를 실행합니다.

### 첫 실행
처음 실행 시 AI 서비스는 백그라운드에서 언어 모델 (Qwen2.5-0.5B)을 로드합니다. 진행 상황은 오른쪽 하단의 Eliot 채팅 위젯에서 확인할 수 있습니다. 하드웨어에 따라 **1~5분**이 소요됩니다.

> **참고:** 앱은 즉시 사용 가능합니다 – 진행 바가 100%가 되면 Eliot이 활성화됩니다.

## 5. SEPGrid – 스프레드시트

SEPGrid는 시트당 **10,000행 × 26열**을 지원하는 강력한 스프레드시트입니다 – Microsoft Excel 및 OpenOffice Calc와 유사합니다.

### 5.1 기본 작업

| 작업 | 설명 |
|------|------|
| 셀 클릭 | 셀 선택 |
| 더블클릭 / F2 | 셀 편집 |
| `Enter` | 확인 후 다음 행으로 |
| `Tab` | 확인 후 다음 열로 |
| `Escape` | 편집 취소 |
| `Ctrl + Z` | 실행 취소 |
| `Ctrl + Y` | 다시 실행 |
| `Ctrl + C` | 복사 |
| `Ctrl + V` | 붙여넣기 |
| `Ctrl + X` | 잘라내기 |
| `Ctrl + B` | 굵게 |
| `Ctrl + I` | 기울임꼴 |
| `Ctrl + U` | 밑줄 |
| `Ctrl + F` | 찾기 및 바꾸기 |
| `Del` | 셀 내용 지우기 |

**다중 선택:** 마우스를 드래그하거나 `Shift + 클릭`.

**채우기 핸들:** 셀 오른쪽 하단의 파란 사각형을 드래그하여 값 또는 수식 복사.

### 5.2 수식 입력줄

```
┌─────────┬────┬───────────────────────────────────────────┐
│   A1    │ fx │  =SUM(A1:A10)                             │
└─────────┴────┴───────────────────────────────────────────┘
  ↑          ↑              ↑
  셀 주소       기호            입력 필드 (원시 수식 표시)
```

- **셀 주소**: 활성 셀 또는 선택 범위 표시 (예: `A1:B5`)
- **기호 fx**: 수식 입력을 나타냄
- **입력 필드**: 항상 원시 수식 표시 – 결과가 아님

**오류 표시:** `#오류`, `#REF!`, `#DIV/0!` 등 수식 오류 시 필드 주위에 빨간 테두리가 표시됩니다.

### 5.3 수식 입력

수식 입력은 항상 `=`로 시작합니다:

```
=SUM(A1:A10)               합계
=AVERAGE(B1:B5)            평균
=MAX(C1:C100)              최댓값
=MIN(D1:D50)               최솟값
=COUNT(A:A)                개수
=IF(A1>0;"예";"아니요")    조건
=VLOOKUP(...)              수직 검색
=ROUND(A1; 2)              반올림
```

#### 마우스 클릭으로 수식 참조

1. `=` 또는 `=SUM(` 같은 함수 입력
2. 원하는 셀 클릭 → 주소 삽입
3. 범위는 드래그 → 범위 삽입됨 (예: `A1:B10`)

### 5.4 시트 간 수식

다른 시트 참조 – Excel과 동일:

```
=Sheet2!A1                    Sheet2의 단일 셀
=SUM(Sheet2!A1:B10)           Sheet2의 합계
=Sheet1!A1 + Sheet2!B5        여러 시트 조합
=AVERAGE(매출!C1:C100)        '매출' 시트의 범위
```

**마우스 클릭으로:**
1. 셀에 `=` 입력
2. 다른 시트 탭 클릭 → `Sheet2!` 삽입
3. 원하는 셀 클릭 → `A1` 추가됨

### 5.5 숫자 서식

도구 모음의 **123 기호**에서:

| 서식 | 예 | 설명 |
|------|------|------|
| 일반 | 1234.5 | 서식 없음 |
| 숫자 | 1,234.56 | 소수점 2자리 서식 |
| 통화 | ₩1,234.56 | 통화 서식 |
| 백분율 | 12.5% | 값×100에 % 기호 추가 |
| 천단위 | 1,234 | 천자리 구분자가 있는 정수 |
| 날짜 | 2026-02-24 | 날짜 서식 |

### 5.6 셀 병합

1. 셀 범위 선택
2. **삽입 → 셀 병합**
3. 왼쪽 상단 셀의 값만 유지됨

**병합 해제:** **삽입 → 셀 병합 해제**

### 5.7 조건부 서식

1. 범위 선택
2. **편집 → 조건부 서식...**
3. 규칙 선택: 보다 큼 / 보다 작음 / 같음 / 사이 / 텍스트 포함
4. 텍스트 및 배경 색상 선택
5. **추가**

**삭제:** 범위 선택 → **편집 → 조건부 서식 제거**

### 5.8 틀 고정

1. 스크롤을 시작할 셀 선택
2. **보기 → 틀 고정**
3. 위의 모든 행과 왼쪽의 열이 고정됨

**틀 고정 해제:** **보기 → 틀 고정 해제**

### 5.9 메모

1. 셀 선택 → **삽입 → 메모 추가** → 텍스트 입력 → 확인
2. 메모가 있는 셀은 오른쪽 상단에 **빨간 삼각형** 표시

### 5.10 찾기 및 바꾸기

`Ctrl + F` 또는 **편집 → 찾기 및 바꾸기**:
- **찾기**: ◀ ▶로 탐색
- **바꾸기**: 하나씩 또는 모두

### 5.11 행과 열 삽입/삭제

**삽입** 메뉴에서: 위/아래에 행 삽입, 왼쪽/오른쪽에 열 삽입, 행/열 삭제.

### 5.12 상태 표시줄

하단에 다중 선택 통계가 자동으로 표시됩니다:

```
Σ 합계: 12,345   ⌀ 평균: 1,234   ↓ 최소: 100   ↑ 최대: 5,000   개수: 10
```

## 6. SEPShow – 프레젠테이션

SEPShow는 **Konva** (캔버스 렌더러)를 기반으로 한 슬라이드 프레젠테이션 편집기입니다.

### 6.1 인터페이스

| 영역 | 설명 |
|------|------|
| **왼쪽 사이드바** | 슬라이드 목록 – 미리보기, 드래그로 순서 변경 |
| **메인 영역** | 활성 슬라이드의 캔버스 편집기 |
| **도구 모음** | 도구, 내보내기, 프레젠테이션 모드 |
| **발표자 노트** | 현재 슬라이드의 노트 |

### 6.2 프레젠테이션 모드

**발표**를 클릭하여 전체화면 프레젠테이션:
- 탐색: 화살표 키 `→` / `←` 또는 클릭
- `Esc`로 프레젠테이션 모드 종료

- **데이터 저장:** 모든 문서는 로컬 브라우저 저장소에 저장됩니다. 외부 서버로 데이터가 전송되지 않습니다.
