## 1. סקירה כללית

SEPOffice היא חבילת תוכנה משרדית לשולחן העבודה הנתמכת בבינה מלאכותית עבור Windows, המורכבת משלושה יישומים משולבים:

| יישום | פונקציה |
|-------|---------|
| **SEPWrite** | יצירת מסמכי טקסט עשיר, ייצוא ל-Word/PDF |
| **SEPGrid** | גיליון אלקטרוני עם נוסחאות וגרפים |
| **SEPShow** | מצגות עם עורך על בד ציור |

שלושת היישומים מחוברים ל-**Eliot** – עוזר בינה מלאכותית מובנה שמנסח טקסטים, מייצר נוסחאות ומעצב שקופיות.

## 2. התקנה והפעלה

### תוכנית ההתקנה
הפעל את `SEPOffice Setup 1.0.1.exe`. האשף ינחה אותך בתהליך ההתקנה:
- תיקיית התקנה ניתנת לבחירה (ברירת מחדל: `Program Files\SEPOffice`)
- יוצר פריט בתפריט התחל
- קיצור דרך אופציונלי בשולחן העבודה

### הפעלה ישירה (ללא התקנה)
הפעל את `release\win-unpacked\SEPOffice.exe`.

### הפעלה ראשונה
בהפעלה הראשונה, שירות הבינה המלאכותית טוען את מודל השפה (Qwen2.5-0.5B) ברקע. ניתן לראות את ההתקדמות בווידג'ט הצ'אט של Eliot בפינה הימנית התחתונה. בהתאם לחומרה, זה לוקח **1–5 דקות**.

> **הערה:** היישום מוכן לשימוש מיד – Eliot מופעל כאשר סרגל ההתקדמות מגיע ל-100%.

## 5. SEPGrid – גיליון אלקטרוני

SEPGrid הוא גיליון אלקטרוני עוצמתי עם **10,000 שורות × 26 עמודות** לגיליון – דומה ל-Microsoft Excel ו-OpenOffice Calc.

### 5.1 שימוש בסיסי

| פעולה | תיאור |
|-------|-------|
| לחיצה על תא | בחירת תא |
| לחיצה כפולה / F2 | עריכת תא |
| `Enter` | אשר, עבור לשורה הבאה |
| `Tab` | אשר, עבור לעמודה הבאה |
| `Escape` | ביטול עריכה |
| `Ctrl + Z` | בטל |
| `Ctrl + Y` | בצע שוב |
| `Ctrl + C` | העתק |
| `Ctrl + V` | הדבק |
| `Ctrl + X` | גזור |
| `Ctrl + B` | מודגש |
| `Ctrl + I` | נטוי |
| `Ctrl + U` | קו תחתון |
| `Ctrl + F` | חפש והחלף |
| `Del` | נקה תוכן תא |

**בחירה מרובה:** גרור בעכבר או `Shift + לחיצה`.

**ידית מילוי:** גרור את הריבוע הכחול בפינה הימנית התחתונה של תא להעתקת ערכים או נוסחאות.

### 5.2 שורת הנוסחאות

```
┌─────────┬────┬───────────────────────────────────────────┐
│   A1    │ fx │  =SUM(A1:A10)                             │
└─────────┴────┴───────────────────────────────────────────┘
  ↑          ↑              ↑
  כתובת תא    סמל             שדה קלט (מציג נוסחה גולמית)
```

- **כתובת תא**: מציג את התא הפעיל או הטווח הנבחר (לדוגמה `A1:B5`)
- **סמל fx**: מציין הזנת נוסחה
- **שדה קלט**: מציג תמיד את הנוסחה הגולמית – לא את התוצאה

**הצגת שגיאות:** בשגיאות נוסחה כמו `#שגיאה`, `#REF!`, `#חלוקה/0!` וכד', השדה מוקף במסגרת אדומה והשגיאה מוצגת משמאל.

### 5.3 הזנת נוסחאות

הזנת נוסחה מתחילה תמיד עם `=`:

```
=SUM(A1:A10)               סכום
=AVERAGE(B1:B5)            ממוצע
=MAX(C1:C100)              מקסימום
=MIN(D1:D50)               מינימום
=COUNT(A:A)                ספירה
=IF(A1>0;"כן";"לא")        תנאי
=VLOOKUP(...)              חיפוש אנכי
=ROUND(A1; 2)              עיגול
```

#### הפניות בנוסחאות בלחיצת עכבר

1. הקלד `=` או פונקציה כמו `=SUM(`
2. לחץ על התא הרצוי ← הכתובת תוכנס
3. גרור לטווחים ← הטווח יוכנס (לדוגמה `A1:B10`)

### 5.4 נוסחאות בין גיליונות

הפניות לגיליונות אחרים – כמו ב-Excel:

```
=Sheet2!A1                    תא בודד מ-Sheet2
=SUM(Sheet2!A1:B10)           סכום מ-Sheet2
=Sheet1!A1 + Sheet2!B5        שילוב ממספר גיליונות
=AVERAGE(Sales!C1:C100)       טווח מגיליון "Sales"
```

**בלחיצת עכבר:**
1. הקלד `=` בתא
2. לחץ על לשונית גיליון אחרת ← מוסיף `Sheet2!`
3. לחץ על התא הרצוי ← מוסיף `A1`

### 5.5 פורמטים של מספרים

דרך סמל **123** בסרגל הכלים:

| פורמט | דוגמה | תיאור |
|-------|-------|-------|
| רגיל | 1234.5 | ללא עיצוב |
| מספר | 1,234.56 | פורמט עם 2 מקומות עשרוניים |
| מטבע | ₪1,234.56 | פורמט מטבע |
| אחוז | 12.5% | ערך × 100 עם סימן % |
| אלפים | 1,234 | מספרים שלמים עם מפריד אלפים |
| תאריך | 24/02/2026 | פורמט תאריך |

### 5.6 מיזוג תאים

1. בחר טווח תאים
2. **הוסף ← מזג תאים**
3. רק ערך התא הימני העליון נשמר

**פיצול:** **הוסף ← פצל תאים**

### 5.7 עיצוב מותנה

1. בחר טווח
2. **ערוך ← עיצוב מותנה...**
3. בחר כלל: גדול מ / קטן מ / שווה ל / בין / הטקסט מכיל
4. בחר צבעי טקסט ורקע
5. **הוסף**

**מחיקה:** בחר טווח ← **ערוך ← מחק עיצוב מותנה**

### 5.8 הקפאת חלוניות

1. בחר את התא שממנו יתחיל הגלילה
2. **תצוגה ← הקפא חלוניות**
3. כל השורות מעליו והעמודות משמאל יוקפאו

**ביטול הקפאה:** **תצוגה ← בטל הקפאת חלוניות**

### 5.9 הערות

1. בחר תא ← **הוסף ← הוסף הערה** ← הקלד טקסט ← אישור
2. תאים עם הערות מציגים **משולש אדום** בפינה הימנית העליונה

### 5.10 חיפוש והחלפה

`Ctrl + F` או **ערוך ← חפש והחלף**:
- **חפש**: ניווט ◀ ▶
- **החלף**: בנפרד או הכל בבת אחת

### 5.11 הוספה/מחיקה של שורות ועמודות

דרך תפריט **הוסף**: הוסף שורה מעל/מתחת, עמודה שמאלה/ימינה, מחק שורה/עמודה.

### 5.12 שורת הסטטוס

בתחתית מוצגות אוטומטית סטטיסטיקות לבחירות מרובות:

```
Σ סכום: 12,345   ⌀ ממוצע: 1,234   ↓ מינ': 100   ↑ מקס': 5,000   ספירה: 10
```

### 5.13 ייבוא / ייצוא

| פונקציה | פורמט | תיאור |
|---------|-------|-------|
| ייבוא | `.xlsx`, `.xlsm` | פתיחת קבצי Excel |
| ייבוא CSV | `.csv` | קבצים מופרדים בפסיק/נקודה-פסיק |
| ייצוא | `.xlsx` | שמירה כקובץ Excel |
| ייצוא CSV | `.csv` | שמירה כ-CSV (UTF-8, נקודה-פסיק) |
| הדפסה / PDF | — | תצוגה מקדימה, שורות ממולאות בלבד |
| הוסף תמונה | PNG, JPG | הטמעת תמונה בגיליון |

### 5.14 פונקציות בינה מלאכותית

#### מחולל נוסחאות ב-AI

> "חשב את הממוצע של כל הערכים החיוביים בעמודה B"

## 6. SEPShow – מצגות

SEPShow הוא עורך מצגות מבוסס שקופיות הבנוי על **Konva** (מנוע רינדור בד ציור).

### 6.1 ממשק

| אזור | תיאור |
|------|-------|
| **סרגל צד ימני** | רשימת שקופיות – תצוגה מקדימה, שינוי סדר בגרירה |
| **אזור ראשי** | עורך בד ציור של השקופית הפעילה |
| **סרגל כלים** | כלים, ייצוא, מצב מצגת |
| **הערות מציג** | הערות לשקופית הנוכחית |

### 6.2 מצב מצגת

לחץ על **הצג** למצגת במסך מלא:
- ניווט: מקשי חצים `→` / `←` או לחיצה
- `Esc` מסיים את מצב המצגת
