# Panduan Pengguna SEPOffice

Selamat datang di **SEPOffice**, suite kantor pintar bertenaga AI Anda. SEPOffice menggabungkan alat produktivitas klasik dengan AI lokal tercanggih untuk membantu Anda bekerja lebih cepat dan lebih pintar.

## 2. SEPWrite - Editor Cerdas
- **Penyelesaian AI**: Mulai mengetik, dan biarkan Eliot menyarankan kata-kata berikutnya.
- **Optimalisasi Nada**: Sorot teks dan minta Eliot untuk membuatnya lebih profesional atau ringkas.

![SEPWrite](../assets/sepwrite.png)

## 3. SEPGrid - Spreadsheet Dinamis
- **Generasi Formula AI**: Cukup jelaskan apa yang Anda inginkan, dan Eliot akan menulis formulanya.
- **Visualisasi**: Buat bagan dan tabel secara instan.

![SEPGrid](../assets/sepgrid.png)

## 4. SEPShow - Presentasi Memukau
- **Magic Layout**: Satu klik para mengatur elemen Anda dengan sempurna.
- **Vibe Check**: Biarkan Eliot meninjau konsistensi visual slide Anda.
