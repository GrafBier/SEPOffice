# SEPOffice ユーザーガイド

**SEPOffice** へようこそ。AIを搭載したインテリジェントなオフィススイートです。SEPOfficeは、従来の生産性ツールと最先端のローカルAIを組み合わせ、より速く、よりスマートな作業をサポートします。

![概要](../assets/overview.png)

---

## 2. SEPWrite - インテリジェントエディター
- **AI補完**: 入力を開始すると、Eliotが次の言葉を提案します。
- **トーンの最適化**: テキストをハイライトして、Eliotによりプロフェッショナル、または簡潔な表現への修正を依頼できます。

![SEPWrite](../assets/sepwrite.png)

---

## 3. SEPGrid - ダイナミックなスプレッドシート
- **AI数式生成**: やりたいことを説明するだけで、Eliotが数式を作成します。
- **視覚化**: グラフやテーブルを即座に作成します。

![SEPGrid](../assets/sepgrid.png)

---

## 4. SEPShow - 魅力的なプレゼンテーション
- **マジックレイアウト**: ワンクリックで要素を完璧に配置します。
- **バイブチェック**: Eliotがスライドの視覚的一貫性をチェックします。

![SEPShow](../assets/sepshow.png)
