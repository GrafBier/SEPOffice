## 1. 概述

SEPOffice 是一款适用于 Windows 的 AI 辅助桌面办公套件，由三个集成应用程序组成：

| 应用 | 功能 |
|-----|------|
| **SEPWrite** | 创建富文本文档，导出为 Word/PDF |
| **SEPGrid** | 带公式和图表的电子表格 |
| **SEPShow** | 带画布编辑器的演示文稿 |

所有三个应用都连接到 **Eliot** – 一个内置的 AI 助手，可以起草文本、生成公式和设计幻灯片。

## 2. 安装与启动

### 安装程序
运行 `SEPOffice Setup 1.0.1.exe`。向导将指导您完成安装：
- 可选安装文件夹（默认：`Program Files\SEPOffice`）
- 创建开始菜单项
- 可选的桌面快捷方式

### 直接启动（无需安装）
运行 `release\win-unpacked\SEPOffice.exe`。

### 首次启动
首次启动时，AI 服务在后台加载语言模型（Qwen2.5-0.5B）。进度显示在右下角的 Eliot 聊天部件中。根据硬件配置，这需要 **1–5 分钟**。

> **注意：** 应用程序立即可用 – 当进度条达到 100% 时，Eliot 将激活。

## 5. SEPGrid – 电子表格

SEPGrid 是一个功能强大的电子表格，每个工作表有 **10,000 行 × 26 列** – 类似于 Microsoft Excel 和 OpenOffice Calc。

### 5.1 基本操作

| 操作 | 说明 |
|------|------|
| 点击单元格 | 选择单元格 |
| 双击 / F2 | 编辑单元格 |
| `Enter` | 确认，移至下一行 |
| `Tab` | 确认，移至下一列 |
| `Escape` | 取消编辑 |
| `Ctrl + Z` | 撤销 |
| `Ctrl + Y` | 重做 |
| `Ctrl + C` | 复制 |
| `Ctrl + V` | 粘贴 |
| `Ctrl + X` | 剪切 |
| `Ctrl + B` | 粗体 |
| `Ctrl + I` | 斜体 |
| `Ctrl + U` | 下划线 |
| `Ctrl + F` | 查找与替换 |
| `Del` | 清除单元格内容 |

**多选：** 拖动鼠标或 `Shift + 点击`。

**填充柄：** 拖动单元格右下角的蓝色方块，复制值或公式。

### 5.2 公式栏

```
┌─────────┬────┬───────────────────────────────────────────┐
│   A1    │ fx │  =SUM(A1:A10)                             │
└─────────┴────┴───────────────────────────────────────────┘
  ↑          ↑              ↑
  单元格地址   符号             输入栏（显示原始公式）
```

- **单元格地址**：显示活动单元格或所选区域（例如 `A1:B5`）
- **符号 fx**：表示公式输入
- **输入栏**：始终显示原始公式 – 而非结果

**错误显示：** 出现 `#错误`、`#REF!`、`#DIV/0!` 等公式错误时，栏位以红色边框标示，错误显示在右侧。

### 5.3 输入公式

公式输入始终以 `=` 开头：

```
=SUM(A1:A10)               求和
=AVERAGE(B1:B5)            平均值
=MAX(C1:C100)              最大值
=MIN(D1:D50)               最小值
=COUNT(A:A)                计数
=IF(A1>0;"是";"否")         条件
=VLOOKUP(...)              垂直查找
=ROUND(A1; 2)              四舍五入
```

#### 鼠标点击公式引用

1. 输入 `=` 或函数如 `=SUM(`
2. 点击所需单元格 → 地址将被插入
3. 拖动选择区域 → 区域将被插入（例如 `A1:B10`）

### 5.4 跨工作表公式

引用其他工作表 – 与 Excel 一样：

```
=Sheet2!A1                    Sheet2 中的单个单元格
=SUM(Sheet2!A1:B10)           Sheet2 的求和
=Sheet1!A1 + Sheet2!B5        多工作表组合
=AVERAGE(销售!C1:C100)        来自"销售"工作表的区域
```

**通过鼠标点击：**
1. 在单元格中输入 `=`
2. 点击另一个工作表标签 → 插入 `Sheet2!`
3. 点击所需单元格 → 添加 `A1`

### 5.5 数字格式

通过工具栏中的 **123 符号**：

| 格式 | 示例 | 说明 |
|------|------|------|
| 标准 | 1234.5 | 无格式 |
| 数字 | 1,234.56 | 带2位小数的格式 |
| 货币 | ¥1,234.56 | 货币格式 |
| 百分比 | 12.5% | 值×100加%符号 |
| 千位 | 1,234 | 带千位分隔符的整数 |
| 日期 | 2026-02-24 | 日期格式 |

### 5.6 合并单元格

1. 选择单元格区域
2. **插入 → 合并单元格**
3. 仅保留左上角单元格的值

**取消合并：** **插入 → 取消合并单元格**

### 5.7 条件格式

1. 选择区域
2. **编辑 → 条件格式...**
3. 选择规则：大于 / 小于 / 等于 / 介于 / 文本包含
4. 选择文字和背景颜色
5. **添加**

**删除：** 选择区域 → **编辑 → 删除条件格式**

### 5.8 冻结窗格

1. 选择滚动起始单元格
2. **视图 → 冻结窗格**
3. 该行以上的所有行和左侧列将被冻结

**取消冻结：** **视图 → 取消冻结窗格**

### 5.9 批注

1. 选择单元格 → **插入 → 添加批注** → 输入文字 → 确定
2. 有批注的单元格在右上角显示**红色三角形**

### 5.10 查找与替换

`Ctrl + F` 或 **编辑 → 查找与替换**：
- **查找**：◀ ▶ 导航
- **替换**：逐个或全部替换

### 5.11 插入/删除行和列

通过 **插入** 菜单：在上方/下方插入行，在左侧/右侧插入列，删除行/列。

### 5.12 状态栏

底部自动显示多选的统计信息：

```
Σ 求和: 12,345   ⌀ 平均: 1,234   ↓ 最小: 100   ↑ 最大: 5,000   计数: 10
```

### 5.13 导入/导出

| 功能 | 格式 | 说明 |
|------|------|------|
| 导入 | `.xlsx`, `.xlsm` | 打开 Excel 文件 |
| CSV 导入 | `.csv` | 逗号/分号分隔文件 |
| 导出 | `.xlsx` | 保存为 Excel 文件 |
| CSV 导出 | `.csv` | 保存为 CSV（UTF-8，分号） |
| 打印 / PDF | — | 打印预览，仅含有数据的行 |
| 插入图片 | PNG, JPG | 在表格中嵌入图片 |

### 5.14 AI 功能

#### AI 公式生成器
点击公式栏旁的 **✨**：
> "计算 B 列所有正值的平均值"

## 6. SEPShow – 演示文稿

SEPShow 是基于 **Konva**（画布渲染器）构建的幻灯片演示编辑器。

### 6.1 界面

| 区域 | 说明 |
|------|------|
| **左侧边栏** | 幻灯片列表 – 预览，通过拖拽更改顺序 |
| **主区域** | 活动幻灯片的画布编辑器 |
| **工具栏** | 工具、导出、演示模式 |
| **演讲者备注** | 当前幻灯片的备注 |

### 6.2 演示模式

点击 **演示** 进行全屏演示：
- 导航：方向键 `→` / `←` 或点击
- `Esc` 退出演示模式

## 技术说明

- **数据存储：** 所有文档均存储在本地浏览器存储中。不向外部服务器发送任何数据。
- **AI 模型：** Qwen2.5-0.5B 完全本地运行 – AI 功能不需要网络连接。
- **系统要求：** Windows 10/11，最小 4 GB RAM（AI 模型建议 8 GB）
