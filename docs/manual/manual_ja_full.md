## 1. 概要

SEPOffice は、Windows 向けの AI 搭載デスクトップオフィススイートで、3 つの統合アプリケーションで構成されています：

| アプリ | 機能 |
|--------|------|
| **SEPWrite** | リッチテキスト文書の作成、Word/PDF へのエクスポート |
| **SEPGrid** | 数式とグラフを備えたスプレッドシート |
| **SEPShow** | キャンバスエディターを使ったプレゼンテーション |

3 つすべてのアプリは **Eliot** に接続されています – テキストを作成し、数式を生成し、スライドをデザインする内蔵 AI アシスタントです。

## 2. インストールと起動

### インストーラー
`SEPOffice Setup 1.0.1.exe` を実行します。ウィザードがインストール手順を案内します：
- インストールフォルダーを選択可能（デフォルト：`Program Files\SEPOffice`）
- スタートメニュー項目を作成
- デスクトップショートカット（オプション）

### 直接起動（インストールなし）
`release\win-unpacked\SEPOffice.exe` を実行します。

### 初回起動
初回起動時、AI サービスはバックグラウンドで言語モデル（Qwen2.5-0.5B）を読み込みます。進行状況は右下の Eliot チャットウィジェットで確認できます。ハードウェアによって **1〜5 分**かかります。

> **注意：** アプリはすぐに使用可能です – プログレスバーが 100% になると Eliot がアクティブになります。

## 5. SEPGrid – スプレッドシート

SEPGrid は、シートごとに **10,000 行 × 26 列**を持つ強力なスプレッドシートです – Microsoft Excel や OpenOffice Calc に類似しています。

### 5.1 基本操作

| 操作 | 説明 |
|------|------|
| セルをクリック | セルを選択 |
| ダブルクリック / F2 | セルを編集 |
| `Enter` | 確定、次の行へ |
| `Tab` | 確定、次の列へ |
| `Escape` | 編集をキャンセル |
| `Ctrl + Z` | 元に戻す |
| `Ctrl + Y` | やり直す |
| `Ctrl + C` | コピー |
| `Ctrl + V` | 貼り付け |
| `Ctrl + X` | 切り取り |
| `Ctrl + B` | 太字 |
| `Ctrl + I` | 斜体 |
| `Ctrl + U` | 下線 |
| `Ctrl + F` | 検索と置換 |
| `Del` | セルの内容をクリア |

**複数選択：** マウスをドラッグするか `Shift + クリック`。

**フィルハンドル：** セルの右下隅の青い四角をドラッグして値または数式をコピー。

### 5.2 数式バー

```
┌─────────┬────┬───────────────────────────────────────────┐
│   A1    │ fx │  =SUM(A1:A10)                             │
└─────────┴────┴───────────────────────────────────────────┘
  ↑          ↑              ↑
  セル番地      シンボル          入力フィールド（生の数式を表示）
```

- **セル番地**：アクティブなセルまたは選択範囲を表示（例：`A1:B5`）
- **シンボル fx**：数式入力を示す
- **入力フィールド**：常に生の数式を表示 – 結果ではない

**エラー表示：** `#エラー`、`#REF!`、`#DIV/0!` などの数式エラー時、フィールドが赤いボーダーで囲まれ、右側にエラーが表示されます。

### 5.3 数式の入力

数式入力は常に `=` で始まります：

```
=SUM(A1:A10)               合計
=AVERAGE(B1:B5)            平均
=MAX(C1:C100)              最大値
=MIN(D1:D50)               最小値
=COUNT(A:A)                カウント
=IF(A1>0;"はい";"いいえ")   条件
=VLOOKUP(...)              垂直検索
=ROUND(A1; 2)              四捨五入
```

#### マウスクリックによる数式参照

1. `=` または `=SUM(` などの関数を入力
2. 目的のセルをクリック → アドレスが挿入される
3. 範囲はドラッグ → 範囲が挿入される（例：`A1:B10`）

### 5.4 シート間の数式

他のシートへの参照 – Excel と同様：

```
=Sheet2!A1                    Sheet2 の単一セル
=SUM(Sheet2!A1:B10)           Sheet2 の合計
=Sheet1!A1 + Sheet2!B5        複数シートの組み合わせ
=AVERAGE(売上!C1:C100)        「売上」シートからの範囲
```

**マウスクリックで：**
1. セルに `=` を入力
2. 別のシートタブをクリック → `Sheet2!` が挿入される
3. 目的のセルをクリック → `A1` が追加される

### 5.5 数値フォーマット

ツールバーの **123 シンボル**から：

| フォーマット | 例 | 説明 |
|-------------|------|------|
| 標準 | 1234.5 | フォーマットなし |
| 数値 | 1,234.56 | 小数点2桁のフォーマット |
| 通貨 | ¥1,234.56 | 通貨フォーマット |
| パーセント | 12.5% | 値×100に%記号を付加 |
| 千単位 | 1,234 | 千位区切り付き整数 |
| 日付 | 2026/02/24 | 日付フォーマット |

### 5.6 セルの結合

1. セル範囲を選択
2. **挿入 → セルを結合**
3. 左上のセルの値のみ保持される

**結合解除：** **挿入 → セルの結合を解除**

### 5.7 条件付き書式

1. 範囲を選択
2. **編集 → 条件付き書式...**
3. ルールを選択：より大きい / より小さい / 等しい / の間 / テキストを含む
4. テキストと背景の色を選択
5. **追加**

**削除：** 範囲を選択 → **編集 → 条件付き書式を削除**

### 5.8 ウィンドウ枠の固定

1. スクロールを開始するセルを選択
2. **表示 → ウィンドウ枠を固定**
3. すべての上の行と左の列が固定される

**固定解除：** **表示 → ウィンドウ枠の固定を解除**

### 5.9 コメント

1. セルを選択 → **挿入 → コメントを追加** → テキストを入力 → OK
2. コメントのあるセルは右上隅に**赤い三角**が表示される

### 5.10 検索と置換

`Ctrl + F` または **編集 → 検索と置換**：
- **検索**：◀ ▶ でナビゲート
- **置換**：1つずつまたは全て

### 5.11 行と列の挿入/削除

**挿入**メニューから：上/下に行を挿入、左/右に列を挿入、行/列を削除。

### 5.12 ステータスバー

下部に複数選択の統計が自動表示されます：

```
Σ 合計: 12,345   ⌀ 平均: 1,234   ↓ 最小: 100   ↑ 最大: 5,000   件数: 10
```

### 5.13 インポート / エクスポート

| 機能 | フォーマット | 説明 |
|------|------------|------|
| インポート | `.xlsx`, `.xlsm` | Excel ファイルを開く |
| CSV インポート | `.csv` | カンマ/セミコロン区切りファイル |
| エクスポート | `.xlsx` | Excel ファイルとして保存 |
| CSV エクスポート | `.csv` | CSV として保存（UTF-8、セミコロン） |
| 印刷 / PDF | — | 印刷プレビュー、入力行のみ |
| 画像の挿入 | PNG, JPG | スプレッドシートに画像を埋め込む |

### 5.14 AI 機能

#### AI 数式ジェネレーター
数式バーの **✨** をクリック：
> "B列のすべての正の値の平均を計算して"
> → `=AVERAGEIF(B:B;">0")`

#### AI テーブルアシスタント
> "2025年の月次売上テーブルを作成して。列：月、収益、費用、利益"

## 6. SEPShow – プレゼンテーション

SEPShow は **Konva**（キャンバスレンダラー）をベースにしたスライドプレゼンテーションエディターです。

### 6.1 インターフェース

| エリア | 説明 |
|--------|------|
| **左サイドバー** | スライドリスト – プレビュー、ドラッグで順序変更 |
| **メインエリア** | アクティブなスライドのキャンバスエディター |
| **ツールバー** | ツール、エクスポート、プレゼンモード |
| **発表者ノート** | 現在のスライドのノート |

### 6.2 プレゼンテーションモード

**発表**をクリックしてフルスクリーンプレゼンテーション：
- ナビゲーション：矢印キー `→` / `←` またはクリック
- `Esc` でプレゼンテーションモードを終了

## 技術的な注記

- **データ保存：** すべての文書はローカルのブラウザストレージに保存されます。外部サーバーにデータは送信されません。
- **AI モデル：** Qwen2.5-0.5B は完全にローカルで実行されます – AI 機能にインターネット接続は不要です。
- **システム要件：** Windows 10/11、最小 4 GB RAM（AI モデルには 8 GB 推奨）
