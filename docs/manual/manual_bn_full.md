## 1. সংক্ষিপ্ত বিবরণ

SEPOffice হল Windows-এর জন্য একটি AI-চালিত ডেস্কটপ অফিস স্যুট যাতে তিনটি সমন্বিত অ্যাপ্লিকেশন রয়েছে:

| অ্যাপ | কার্যকারিতা |
|-------|------------|
| **SEPWrite** | রিচ টেক্সট ডকুমেন্ট তৈরি, Word/PDF রপ্তানি |
| **SEPGrid** | সূত্র ও চার্ট সহ স্প্রেডশিট |
| **SEPShow** | ক্যানভাস এডিটর দিয়ে প্রেজেন্টেশন |

তিনটি অ্যাপই **Eliot**-এর সাথে সংযুক্ত – একটি অন্তর্নির্মিত AI সহায়ক যা টেক্সট লেখে, সূত্র তৈরি করে এবং স্লাইড ডিজাইন করে।

## 2. ইনস্টলেশন ও শুরু

### ইনস্টলার
`SEPOffice Setup 1.0.1.exe` চালান। উইজার্ড ইনস্টলেশন ধাপগুলি নির্দেশ করবে:
- ইনস্টলেশন ফোল্ডার নির্বাচন করা যাবে (ডিফল্ট: `Program Files\SEPOffice`)
- স্টার্ট মেনু আইটেম তৈরি করুন
- ডেস্কটপ শর্টকাট (ঐচ্ছিক)

### সরাসরি চালান (ইনস্টলেশন ছাড়া)
`release\win-unpacked\SEPOffice.exe` চালান।

### প্রথম শুরু
প্রথমবার চালানোর সময়, AI সার্ভিস ব্যাকগ্রাউন্ডে ভাষা মডেল (Qwen2.5-0.5B) লোড করে। অগ্রগতি ডানদিকে নিচে Eliot চ্যাট উইজেটে দেখা যায়। হার্ডওয়্যারের উপর নির্ভর করে **১–৫ মিনিট** লাগতে পারে।

> **নোট:** অ্যাপ তাৎক্ষণিক ব্যবহারের জন্য প্রস্তুত – প্রগ্রেস বার ১০০% হলে Eliot সক্রিয় হয়।

## 5. SEPGrid – স্প্রেডশিট

SEPGrid হল একটি শক্তিশালী স্প্রেডশিট যাতে প্রতি শিটে **১০,০০০ সারি × ২৬ কলাম** রয়েছে – Microsoft Excel ও OpenOffice Calc-এর অনুরূপ।

### 5.1 মূল অপারেশন

| ক্রিয়া | বিবরণ |
|--------|-------|
| সেল ক্লিক | সেল নির্বাচন |
| ডাবল-ক্লিক / F2 | সেল সম্পাদনা |
| `Enter` | নিশ্চিত করুন, পরের সারি |
| `Tab` | নিশ্চিত করুন, পরের কলাম |
| `Escape` | সম্পাদনা বাতিল |
| `Ctrl + Z` | পূর্বাবস্থায় ফিরুন |
| `Ctrl + Y` | পুনরায় করুন |
| `Ctrl + C` | কপি |
| `Ctrl + V` | পেস্ট |
| `Ctrl + X` | কাট |
| `Ctrl + B` | বোল্ড |
| `Ctrl + I` | ইটালিক |
| `Ctrl + U` | আন্ডারলাইন |
| `Ctrl + F` | খুঁজুন ও প্রতিস্থাপন করুন |
| `Del` | সেলের বিষয়বস্তু মুছুন |

**বহু-নির্বাচন:** মাউস ড্র্যাগ করুন বা `Shift + ক্লিক`।

**ফিল হ্যান্ডল:** মান বা সূত্র কপি করতে সেলের নিচের ডান কোণে নীল বর্গক্ষেত্র ড্র্যাগ করুন।

### 5.2 সূত্র বার

```
┌─────────┬────┬───────────────────────────────────────────┐
│   A1    │ fx │  =SUM(A1:A10)                             │
└─────────┴────┴───────────────────────────────────────────┘
  ↑          ↑              ↑
  সেল ঠিকানা   প্রতীক           ইনপুট ফিল্ড (কাঁচা সূত্র দেখায়)
```

- **সেল ঠিকানা**: সক্রিয় সেল বা নির্বাচন দেখায় (যেমন: `A1:B5`)
- **প্রতীক fx**: সূত্র ইনপুট নির্দেশ করে
- **ইনপুট ফিল্ড**: সবসময় কাঁচা সূত্র দেখায় – ফলাফল নয়

**ত্রুটি প্রদর্শন:** `#ত্রুটি`, `#REF!`, `#DIV/0!` ইত্যাদি সূত্র ত্রুটিতে ফিল্ডের চারপাশে লাল বর্ডার দেখা যায়।

### 5.3 সূত্র প্রবেশ করানো

সূত্র ইনপুট সবসময় `=` দিয়ে শুরু হয়:

```
=SUM(A1:A10)               যোগফল
=AVERAGE(B1:B5)            গড়
=MAX(C1:C100)              সর্বোচ্চ
=MIN(D1:D50)               সর্বনিম্ন
=COUNT(A:A)                গণনা
=IF(A1>0;"হ্যাঁ";"না")    শর্ত
=VLOOKUP(...)              উল্লম্ব অনুসন্ধান
=ROUND(A1; 2)              বৃত্তাকার
```

#### মাউস ক্লিক দিয়ে সূত্র রেফারেন্স

1. `=` বা `=SUM(` এর মতো ফাংশন টাইপ করুন
2. ইচ্ছুক সেল ক্লিক করুন → ঠিকানা সন্নিবেশিত হয়
3. পরিসরের জন্য ড্র্যাগ করুন → পরিসর সন্নিবেশিত হয় (যেমন: `A1:B10`)

### 5.4 শিট জুড়ে সূত্র

অন্য শিটের রেফারেন্স – Excel-এর মতো:

```
=Sheet2!A1                    Sheet2-এর একক সেল
=SUM(Sheet2!A1:B10)           Sheet2-এর যোগফল
=Sheet1!A1 + Sheet2!B5        একাধিক শিটের সমন্বয়
=AVERAGE(বিক্রয়!C1:C100)     'বিক্রয়' শিটের পরিসর
```

**মাউস ক্লিক দিয়ে:**
1. সেলে `=` টাইপ করুন
2. অন্য শিট ট্যাব ক্লিক করুন → `Sheet2!` সন্নিবেশিত হয়
3. ইচ্ছুক সেল ক্লিক করুন → `A1` যোগ হয়

### 5.5 সংখ্যা বিন্যাস

টুলবারে **123 প্রতীক** থেকে:

| বিন্যাস | উদাহরণ | বিবরণ |
|---------|--------|-------|
| সাধারণ | 1234.5 | কোনো বিন্যাস নেই |
| সংখ্যা | 1,234.56 | ২ দশমিক স্থান বিন্যাস |
| মুদ্রা | ৳1,234.56 | মুদ্রা বিন্যাস |
| শতাংশ | 12.5% | মান×100-এ % চিহ্ন |
| হাজার | 1,234 | হাজার বিভাজক সহ পূর্ণসংখ্যা |
| তারিখ | 24/02/2026 | তারিখ বিন্যাস |

### 5.6 সেল মার্জ করা

1. সেল পরিসর নির্বাচন করুন
2. **সন্নিবেশ > সেল মার্জ করুন**
3. শুধুমাত্র উপরের-বামের সেলের মান রাখা হবে

**মার্জ বাতিল করুন:** **সন্নিবেশ > সেল মার্জ বাতিল**

### 5.7 শর্তাধীন বিন্যাস

1. পরিসর নির্বাচন করুন
2. **সম্পাদনা > শর্তাধীন বিন্যাস...**
3. নিয়ম নির্বাচন করুন: এর চেয়ে বড় / এর চেয়ে ছোট / সমান / মধ্যে / টেক্সট ধারণ করে
4. টেক্সট ও ব্যাকগ্রাউন্ড রঙ নির্বাচন করুন
5. **যোগ করুন**

**মুছুন:** পরিসর নির্বাচন → **সম্পাদনা > শর্তাধীন বিন্যাস মুছুন**

### 5.8 সারি/কলাম স্থির করা

1. যে সেল থেকে স্ক্রোলিং শুরু করতে হবে সেটি নির্বাচন করুন
2. **দেখুন > সারি/কলাম স্থির করুন**
3. উপরের সমস্ত সারি ও বামের কলামগুলি স্থির হয়

**স্থির বাতিল করুন:** **দেখুন > সারি/কলাম স্থির বাতিল**

### 5.9 মন্তব্য

1. সেল নির্বাচন → **সন্নিবেশ > মন্তব্য যোগ করুন** → টেক্সট দিন → ঠিক আছে
2. মন্তব্য সহ সেলের উপরের ডান কোণে **লাল ত্রিভুজ** দেখা যায়

### 5.10 খুঁজুন ও প্রতিস্থাপন করুন

`Ctrl + F` বা **সম্পাদনা > খুঁজুন ও প্রতিস্থাপন করুন**:
- **খুঁজুন**: ◀ ▶ দিয়ে নেভিগেট করুন
- **প্রতিস্থাপন**: একটি একটি করে বা সবগুলো

### 5.11 সারি ও কলাম সন্নিবেশ/মুছুন

**সন্নিবেশ** মেনু থেকে: উপরে/নিচে সারি সন্নিবেশ, বামে/ডানে কলাম সন্নিবেশ, সারি/কলাম মুছুন।

### 5.12 স্ট্যাটাস বার

নিচে বহু-নির্বাচন পরিসংখ্যান স্বয়ংক্রিয়ভাবে দেখায়:

```
Σ যোগ: 12,345   ⌀ গড়: 1,234   ↓ সর্বনিম্ন: 100   ↑ সর্বোচ্চ: 5,000   গণনা: 10
```

### 5.13 আমদানি / রপ্তানি

| কার্যকারিতা | বিন্যাস | বিবরণ |
|------------|---------|-------|
| আমদানি | `.xlsx`, `.xlsm` | Excel ফাইল খুলুন |
| CSV আমদানি | `.csv` | কমা/সেমিকোলন-বিভক্ত ফাইল |
| রপ্তানি | `.xlsx` | Excel ফাইল হিসেবে সংরক্ষণ |
| CSV রপ্তানি | `.csv` | CSV হিসেবে সংরক্ষণ (UTF-8, সেমিকোলন) |
| প্রিন্ট / PDF | — | প্রিন্ট প্রিভিউ, শুধু পূর্ণ সারি |
| ছবি সন্নিবেশ | PNG, JPG | স্প্রেডশিটে ছবি এম্বেড করুন |

### 5.14 AI কার্যকারিতা

#### AI সূত্র জেনারেটর
সূত্র বারে **✨** ক্লিক করুন:
> "B কলামের সমস্ত ধনাত্মক মানের গড় হিসাব করুন"
> → `=AVERAGEIF(B:B;">0")`

#### AI টেবিল সহায়ক
