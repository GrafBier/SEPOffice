## 1. ภาพรวม

SEPOffice คือชุดโปรแกรมสำนักงานเดสก์ท็อปที่ขับเคลื่อนด้วย AI สำหรับ Windows ประกอบด้วยแอปพลิเคชันแบบบูรณาการสามตัว:

| แอป | ฟังก์ชัน |
|-----|---------|
| **SEPWrite** | สร้างเอกสารข้อความสมบูรณ์ ส่งออกเป็น Word/PDF |
| **SEPGrid** | สเปรดชีตพร้อมสูตรและแผนภูมิ |
| **SEPShow** | การนำเสนอด้วยโปรแกรมแก้ไขผ้าใบ |

แอปทั้งสามเชื่อมต่อกับ **Eliot** – ผู้ช่วย AI ในตัวที่เขียนข้อความ สร้างสูตร และออกแบบสไลด์

## 2. การติดตั้งและการเริ่มต้น

### ตัวติดตั้ง
รันไฟล์ `SEPOffice Setup 1.0.1.exe` วิซาร์ดจะแนะนำขั้นตอนการติดตั้ง:
- สามารถเลือกโฟลเดอร์ติดตั้งได้ (ค่าเริ่มต้น: `Program Files\SEPOffice`)
- สร้างรายการเมนูเริ่มต้น
- ทางลัดบนเดสก์ท็อป (ไม่บังคับ)

### รันโดยตรง (ไม่ต้องติดตั้ง)
รันไฟล์ `release\win-unpacked\SEPOffice.exe`

### การเริ่มต้นครั้งแรก
เมื่อรันครั้งแรก บริการ AI จะโหลดโมเดลภาษา (Qwen2.5-0.5B) ในเบื้องหลัง ความคืบหน้าจะแสดงในวิดเจ็ตแชท Eliot ที่ด้านล่างขวา ขึ้นอยู่กับฮาร์ดแวร์ อาจใช้เวลา **1–5 นาที**

> **หมายเหตุ:** แอปพร้อมใช้งานทันที – Eliot จะเปิดใช้งานเมื่อแถบความคืบหน้าถึง 100%

## 5. SEPGrid – สเปรดชีต

SEPGrid คือสเปรดชีตที่ทรงพลังพร้อม **10,000 แถว × 26 คอลัมน์** ต่อชีต – คล้ายกับ Microsoft Excel และ OpenOffice Calc

### 5.1 การดำเนินการพื้นฐาน

| การดำเนินการ | คำอธิบาย |
|------------|---------|
| คลิกเซลล์ | เลือกเซลล์ |
| ดับเบิลคลิก / F2 | แก้ไขเซลล์ |
| `Enter` | ยืนยัน ไปแถวถัดไป |
| `Tab` | ยืนยัน ไปคอลัมน์ถัดไป |
| `Escape` | ยกเลิกการแก้ไข |
| `Ctrl + Z` | เลิกทำ |
| `Ctrl + Y` | ทำซ้ำ |
| `Ctrl + C` | คัดลอก |
| `Ctrl + V` | วาง |
| `Ctrl + X` | ตัด |
| `Ctrl + B` | ตัวหนา |
| `Ctrl + I` | ตัวเอียง |
| `Ctrl + U` | ขีดเส้นใต้ |
| `Ctrl + F` | ค้นหาและแทนที่ |
| `Del` | ล้างเนื้อหาเซลล์ |

**เลือกหลายรายการ:** ลากเมาส์หรือ `Shift + คลิก`

**จุดจับเติม:** ลากสี่เหลี่ยมสีน้ำเงินที่มุมขวาล่างของเซลล์เพื่อคัดลอกค่าหรือสูตร

### 5.2 แถบสูตร

```
┌─────────┬────┬───────────────────────────────────────────┐
│   A1    │ fx │  =SUM(A1:A10)                             │
└─────────┴────┴───────────────────────────────────────────┘
  ↑          ↑              ↑
  ที่อยู่เซลล์   สัญลักษณ์        ช่องป้อนข้อมูล (แสดงสูตรดิบ)
```

- **ที่อยู่เซลล์**: แสดงเซลล์ที่ใช้งานอยู่หรือช่วงที่เลือก (เช่น: `A1:B5`)
- **สัญลักษณ์ fx**: บ่งบอกการป้อนสูตร
- **ช่องป้อนข้อมูล**: แสดงสูตรดิบเสมอ – ไม่ใช่ผลลัพธ์

**การแสดงข้อผิดพลาด:** สำหรับข้อผิดพลาดในสูตรเช่น `#ERROR`, `#REF!`, `#DIV/0!` ช่องจะมีเส้นขอบสีแดงล้อมรอบ

### 5.3 การป้อนสูตร

การป้อนสูตรเริ่มต้นด้วย `=` เสมอ:

```
=SUM(A1:A10)               ผลรวม
=AVERAGE(B1:B5)            ค่าเฉลี่ย
=MAX(C1:C100)              ค่าสูงสุด
=MIN(D1:D50)               ค่าต่ำสุด
=COUNT(A:A)                นับจำนวน
=IF(A1>0;"ใช่";"ไม่ใช่")   เงื่อนไข
=VLOOKUP(...)              การค้นหาแนวตั้ง
=ROUND(A1; 2)              การปัดเศษ
```

#### การอ้างอิงสูตรด้วยการคลิกเมาส์

1. พิมพ์ `=` หรือฟังก์ชันเช่น `=SUM(`
2. คลิกเซลล์ที่ต้องการ → ที่อยู่จะถูกแทรก
3. ลากสำหรับช่วง → ช่วงจะถูกแทรก (เช่น: `A1:B10`)

### 5.4 สูตรข้ามชีต

การอ้างอิงชีตอื่น – เหมือนกับ Excel:

```
=Sheet2!A1                    เซลล์เดียวจาก Sheet2
=SUM(Sheet2!A1:B10)           ผลรวมจาก Sheet2
=Sheet1!A1 + Sheet2!B5        รวมหลายชีต
=AVERAGE(ยอดขาย!C1:C100)      ช่วงจากชีต 'ยอดขาย'
```

**ด้วยการคลิกเมาส์:**
1. พิมพ์ `=` ในเซลล์
2. คลิกแท็บชีตอื่น → `Sheet2!` จะถูกแทรก
3. คลิกเซลล์ที่ต้องการ → `A1` จะถูกเพิ่ม

### 5.5 รูปแบบตัวเลข

จาก **สัญลักษณ์ 123** ในแถบเครื่องมือ:

| รูปแบบ | ตัวอย่าง | คำอธิบาย |
|--------|---------|---------|
| ทั่วไป | 1234.5 | ไม่มีรูปแบบ |
| ตัวเลข | 1,234.56 | รูปแบบทศนิยม 2 ตำแหน่ง |
| สกุลเงิน | ฿1,234.56 | รูปแบบสกุลเงิน |
| เปอร์เซ็นต์ | 12.5% | ค่า×100 พร้อมสัญลักษณ์ % |
| พัน | 1,234 | จำนวนเต็มพร้อมตัวแยกหลักพัน |
| วันที่ | 24/02/2026 | รูปแบบวันที่ |

### 5.6 การผสานเซลล์

1. เลือกช่วงเซลล์
2. **แทรก → ผสานเซลล์**
3. เก็บเฉพาะค่าของเซลล์บนซ้าย

**ยกเลิกการผสาน:** **แทรก → ยกเลิกการผสานเซลล์**

### 5.7 การจัดรูปแบบตามเงื่อนไข

1. เลือกช่วง
2. **แก้ไข → การจัดรูปแบบตามเงื่อนไข...**
3. เลือกกฎ: มากกว่า / น้อยกว่า / เท่ากับ / ระหว่าง / มีข้อความ
4. เลือกสีข้อความและพื้นหลัง
5. **เพิ่ม**

**ลบ:** เลือกช่วง → **แก้ไข → ลบการจัดรูปแบบตามเงื่อนไข**

### 5.8 การตรึงแถว/คอลัมน์

1. เลือกเซลล์ที่จะเริ่มเลื่อน
2. **มุมมอง → ตรึงแถว/คอลัมน์**
3. แถวทั้งหมดด้านบนและคอลัมน์ทางซ้ายจะถูกตรึง

**ยกเลิกการตรึง:** **มุมมอง → ยกเลิกการตรึง**

### 5.9 ความคิดเห็น

1. เลือกเซลล์ → **แทรก → เพิ่มความคิดเห็น** → ป้อนข้อความ → ตกลง
2. เซลล์ที่มีความคิดเห็นจะแสดง **สามเหลี่ยมสีแดง** ที่มุมขวาบน

### 5.10 ค้นหาและแทนที่

`Ctrl + F` หรือ **แก้ไข → ค้นหาและแทนที่**:
- **ค้นหา**: นำทางด้วย ◀ ▶
- **แทนที่**: ทีละรายการหรือทั้งหมด

### 5.11 แทรก/ลบแถวและคอลัมน์

จากเมนู **แทรก**: แทรกแถวด้านบน/ล่าง แทรกคอลัมน์ทางซ้าย/ขวา ลบแถว/คอลัมน์

### 5.12 แถบสถานะ

ด้านล่างจะแสดงสถิติการเลือกหลายรายการโดยอัตโนมัติ:

```
Σ ผลรวม: 12,345   ⌀ ค่าเฉลี่ย: 1,234   ↓ ต่ำสุด: 100   ↑ สูงสุด: 5,000   จำนวน: 10
```

### 5.13 นำเข้า / ส่งออก

| ฟีเจอร์ | รูปแบบ | คำอธิบาย |
|--------|--------|---------|

| นำเข้า CSV | `.csv` | ไฟล์คั่นด้วยจุลภาค/เซมิโคลอน |
| ส่งออก | `.xlsx` | บันทึกเป็นไฟล์ Excel |
| ส่งออก CSV | `.csv` | บันทึกเป็น CSV (UTF-8, เซมิโคลอน) |

| แทรกรูปภาพ | PNG, JPG | ฝังรูปภาพในสเปรดชีต |

## 6. SEPShow – การนำเสนอ
