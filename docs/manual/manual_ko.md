# SEPOffice 사용자 가이드

AI 기반의 지능형 오피스 Suite, **SEPOffice**에 오신 것을 환영합니다. SEPOffice는 기존의 생산성 도구와 최첨단 로컬 AI를 결합하여 더 빠르고 스마트하게 작업할 수 있도록 돕습니다.

![개요](../assets/overview.png)

---

## 2. SEPWrite - 지능형 에디터
- **AI 완성**: 입력을 시작하면 Eliot이 다음 단어를 제안합니다.
- **톤 최적화**: 텍스트를 강조하고 Eliot에게 더 전문적이거나 간결하게 수정하도록 요청하세요.

![SEPWrite](../assets/sepwrite.png)

---

## 3. SEPGrid - 동적 스프레드시트
- **AI 수식 생성**: 원하는 내용을 설명하기만 하면 Eliot이 수식을 작성합니다.
- **시각화**: 즉석에서 차트와 표를 생성합니다.

![SEPGrid](../assets/sepgrid.png)

---

## 4. SEPShow - 매력적인 프레젠테이션
- **매직 레이아웃**: 한 번의 클릭으로 요소를 완벽하게 배치합니다.
- **바이브 체크**: Eliot이 슬라이드의 시각적 일관성을 검토합니다.

![SEPShow](../assets/sepshow.png)
