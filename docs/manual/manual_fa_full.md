## 1. نمای کلی

SEPOffice یک مجموعه نرم‌افزار اداری دسکتاپ با پشتیبانی هوش مصنوعی برای ویندوز است که از سه برنامه یکپارچه تشکیل شده است:

| برنامه | عملکرد |
|--------|---------|
| **SEPWrite** | ایجاد اسناد متنی غنی، صادرکردن به Word/PDF |
| **SEPGrid** | صفحه گسترده با فرمول‌ها و نمودارها |
| **SEPShow** | ارائه‌ها با ویرایشگر بوم |

هر سه برنامه به **Eliot** متصل هستند – دستیار هوش مصنوعی داخلی که متن‌ها را می‌نویسد، فرمول‌ها تولید می‌کند و اسلایدها طراحی می‌کند.

## 2. نصب و راه‌اندازی

### برنامه نصب
فایل `SEPOffice Setup 1.0.1.exe` را اجرا کنید. جادوگر نصب شما را در طول فرایند راهنمایی می‌کند:
- پوشه نصب قابل انتخاب (پیش‌فرض: `Program Files\SEPOffice`)
- ایجاد آیتم منوی شروع
- میانبر اختیاری روی دسکتاپ

### راه‌اندازی مستقیم (بدون نصب)
فایل `release\win-unpacked\SEPOffice.exe` را اجرا کنید.

### اولین راه‌اندازی
در اولین راه‌اندازی، سرویس هوش مصنوعی مدل زبانی (Qwen2.5-0.5B) را در پس‌زمینه بارگذاری می‌کند. پیشرفت در ویجت چت Eliot در گوشه پایین راست قابل مشاهده است. بسته به سخت‌افزار، این فرایند **۱ تا ۵ دقیقه** طول می‌کشد.

> **یادداشت:** برنامه بلافاصله قابل استفاده است – Eliot فعال می‌شود وقتی نوار پیشرفت به ۱۰۰٪ برسد.

## 5. SEPGrid – صفحه گسترده

SEPGrid یک صفحه گسترده قدرتمند با **۱۰,۰۰۰ سطر × ۲۶ ستون** در هر برگ است – مشابه Microsoft Excel و OpenOffice Calc.

### 5.1 استفاده پایه

| عملیات | توضیحات |
|--------|---------|
| کلیک روی سلول | انتخاب سلول |
| دوبار کلیک / F2 | ویرایش سلول |
| `Enter` | تأیید، رفتن به سطر بعدی |
| `Tab` | تأیید، رفتن به ستون بعدی |
| `Escape` | لغو ویرایش |
| `Ctrl + Z` | بازگشت |
| `Ctrl + Y` | تکرار |
| `Ctrl + C` | کپی |
| `Ctrl + V` | جای‌گذاری |
| `Ctrl + X` | برش |
| `Ctrl + B` | ضخیم |
| `Ctrl + I` | مورب |
| `Ctrl + U` | خط زیر |
| `Ctrl + F` | جستجو و جایگزینی |
| `Del` | پاک‌کردن محتوای سلول |

**انتخاب چندگانه:** ماوس را بکشید یا `Shift + کلیک`.

**دستگیره پر کردن:** مربع آبی در گوشه پایین راست سلول را برای کپی مقادیر یا فرمول‌ها بکشید.

### 5.2 نوار فرمول

```
┌─────────┬────┬───────────────────────────────────────────┐
│   A1    │ fx │  =SUM(A1:A10)                             │
└─────────┴────┴───────────────────────────────────────────┘
  ↑          ↑              ↑
  آدرس سلول   نماد             فیلد ورودی (فرمول خام را نشان می‌دهد)
```

- **آدرس سلول**: سلول فعال یا محدوده انتخابی را نشان می‌دهد (مثلاً `A1:B5`)
- **نماد fx**: نشان‌دهنده ورود فرمول است
- **فیلد ورودی**: همیشه فرمول خام را نشان می‌دهد – نه نتیجه را

**نمایش خطا:** هنگام خطاهای فرمول مانند `#خطا`، `#REF!`، `#DIV/0!` و غیره، فیلد با قاب قرمز احاطه می‌شود و خطا در سمت راست نشان داده می‌شود.

### 5.3 وارد کردن فرمول‌ها

ورود فرمول همیشه با `=` شروع می‌شود:

```
=SUM(A1:A10)               جمع
=AVERAGE(B1:B5)            میانگین
=MAX(C1:C100)              حداکثر
=MIN(D1:D50)               حداقل
=COUNT(A:A)                شمارش
=IF(A1>0;"بله";"خیر")      شرط
=VLOOKUP(...)              جستجوی عمودی
=ROUND(A1; 2)              گرد کردن
```

#### مراجع فرمول با کلیک ماوس

1. `=` یا یک تابع مانند `=SUM(` را تایپ کنید
2. روی سلول مورد نظر کلیک کنید ← آدرس درج می‌شود
3. برای محدوده‌ها بکشید ← محدوده درج می‌شود (مثلاً `A1:B10`)

### 5.4 فرمول‌ها بین برگ‌ها

ارجاع به برگ‌های دیگر – مانند Excel:

```
=Sheet2!A1                    سلول واحد از Sheet2
=SUM(Sheet2!A1:B10)           جمع از Sheet2
=Sheet1!A1 + Sheet2!B5        ترکیب از چند برگ
=AVERAGE(Sales!C1:C100)       محدوده از برگ "Sales"
```

**با کلیک ماوس:**
1. `=` را در سلول تایپ کنید
2. روی تب برگ دیگری کلیک کنید ← `Sheet2!` درج می‌شود
3. روی سلول مورد نظر کلیک کنید ← `A1` اضافه می‌شود

### 5.5 قالب‌های اعداد

از طریق نماد **123** در نوار ابزار:

| قالب | مثال | توضیحات |
|------|-----|---------|
| استاندارد | 1234.5 | بدون قالب‌بندی |
| عدد | 1,234.56 | قالب با ۲ رقم اعشار |
| ارز | ۱,۲۳۴.۵۶ ﷼ | قالب ارز |
| درصد | 12.5% | مقدار × ۱۰۰ با علامت % |
| هزارها | 1,234 | اعداد صحیح با جداکننده هزار |
| تاریخ | 1404/12/05 | قالب تاریخ |

### 5.6 ادغام سلول‌ها

1. محدوده سلول‌ها را انتخاب کنید
2. **درج ← ادغام سلول‌ها**
3. فقط مقدار سلول بالا-چپ حفظ می‌شود

**جداکردن:** **درج ← جداکردن سلول‌ها**

### 5.7 قالب‌بندی شرطی

1. محدوده را انتخاب کنید
2. **ویرایش ← قالب‌بندی شرطی...**
3. قانون را انتخاب کنید: بزرگتر از / کوچکتر از / مساوی / بین / متن شامل
4. رنگ‌های متن و پس‌زمینه را انتخاب کنید
5. **اضافه کردن**

**حذف:** محدوده را انتخاب کنید ← **ویرایش ← حذف قالب‌بندی شرطی**

### 5.8 ثابت کردن پنجره‌ها

1. سلولی را انتخاب کنید که اسکرول از آنجا شروع شود
2. **نمایش ← ثابت‌کردن پنجره‌ها**
3. تمام سطرهای بالا و ستون‌های سمت چپ ثابت می‌شوند

**لغو ثابت‌کردن:** **نمایش ← لغو ثابت‌کردن پنجره‌ها**

### 5.9 یادداشت‌ها

1. سلول را انتخاب کنید ← **درج ← اضافه کردن یادداشت** ← متن را وارد کنید ← تأیید
2. سلول‌های دارای یادداشت **مثلث قرمز** در گوشه بالا-راست نشان می‌دهند

### 5.10 جستجو و جایگزینی

`Ctrl + F` یا **ویرایش ← جستجو و جایگزینی**:
- **جستجو**: ناوبری ◀ ▶
- **جایگزینی**: به تنهایی یا همه با هم

### 5.11 درج/حذف سطرها و ستون‌ها

از طریق منوی **درج**: سطر بالا/پایین، ستون چپ/راست را درج کنید، سطر/ستون را حذف کنید.

### 5.12 نوار وضعیت

در پایین، آمار برای انتخاب‌های چندگانه به صورت خودکار نمایش داده می‌شود:

```
Σ جمع: ۱۲,۳۴۵   ⌀ میانگین: ۱,۲۳۴   ↓ کمینه: ۱۰۰   ↑ بیشینه: ۵,۰۰۰   تعداد: ۱۰
```

### 5.13 وارداتکردن / صادرکردن

| عملکرد | قالب | توضیحات |
|--------|-----|---------|
| وارداتکردن | `.xlsx`, `.xlsm` | باز کردن فایل‌های Excel |
| وارداتکردن CSV | `.csv` | فایل‌های جداشده با ویرگول/نقطه‌ویرگول |
| صادرکردن | `.xlsx` | ذخیره به صورت فایل Excel |
| صادرکردن CSV | `.csv` | ذخیره به صورت CSV (UTF-8، نقطه‌ویرگول) |
| چاپ / PDF | — | پیش‌نمایش چاپ، فقط سطرهای پر |
| درج تصویر | PNG, JPG | درج تصویر در صفحه گسترده |

### 5.14 عملکردهای هوش مصنوعی

## 6. SEPShow – ارائه‌ها

SEPShow یک ویرایشگر ارائه مبتنی بر اسلاید است که روی **Konva** (رندرکننده بوم) ساخته شده است.

### 6.1 رابط کاربری

| ناحیه | توضیحات |
|-------|---------|
| **پنل کناری راست** | فهرست اسلایدها – پیش‌نمایش، تغییر ترتیب با کشیدن |
| **ناحیه اصلی** | ویرایشگر بوم اسلاید فعال |
| **نوار ابزار** | ابزارها، صادرکردن، حالت ارائه |
| **یادداشت‌های ارائه‌دهنده** | یادداشت‌ها برای اسلاید فعلی |

### 6.2 حالت ارائه

روی **ارائه** کلیک کنید برای ارائه تمام صفحه:
- ناوبری: کلیدهای جهت `→` / `←` یا کلیک
- `Esc` حالت ارائه را پایان می‌دهد
