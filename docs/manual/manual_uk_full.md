## 1. Огляд

SEPOffice — офісний пакет для настільних комп'ютерів з підтримкою ШІ для Windows, що складається з трьох інтегрованих застосунків:

| Застосунок | Функція |
|-----------|---------|
| **SEPWrite** | Створення документів зі збагаченим текстом, експорт до Word/PDF |
| **SEPGrid** | Електронна таблиця з формулами та графіками |
| **SEPShow** | Презентації з редактором на полотні |

Всі три застосунки пов'язані з **Eliot** – вбудованим ШІ-асистентом, який формулює тексти, генерує формули та проектує слайди.

## 2. Встановлення та Запуск

### Програма Встановлення
Запустіть `SEPOffice Setup 1.0.1.exe`. Майстер проведе вас через встановлення:
- Вибір папки встановлення (за замовчуванням: `Program Files\SEPOffice`)
- Створює елемент меню «Пуск»
- Необов'язковий ярлик на робочому столі

### Пряме Запускання (без встановлення)
Запустіть `release\win-unpacked\SEPOffice.exe`.

### Перший Запуск
При першому запуску служба ШІ завантажує мовну модель (Qwen2.5-0.5B) у фоновому режимі. Прогрес відображається у чат-віджеті Eliot у правому нижньому куті. Залежно від апаратного забезпечення це займає **1–5 хвилин**.

> **Примітка:** Застосунок одразу готовий до використання – Eliot активується, коли панель прогресу досягне 100%.

## 3. Панель Керування

Панель керування є початковою сторінкою SEPOffice.

### Три Плитки Застосунків
- **SEPWrite** – Створити або відкрити документ
- **SEPGrid** – Створити або відкрити таблицю
- **SEPShow** – Створити або відкрити презентацію

### Останні Документи
Під плитками відображаються останні відредаговані документи з піктограмою типу, назвою та датою останньої зміни.

### Навігація

| Елемент | Функція |
|---------|---------|
| **SEPWrite / SEPGrid / SEPShow** | Перемикання між застосунками |
| ⚙️ | Відкрити налаштування |
| ⌨️ | Показати гарячі клавіші |
| 🌙 / ☀️ | Перемикання темного/світлого режиму |

## 5. SEPGrid – Електронна Таблиця

SEPGrid — потужна електронна таблиця з **10 000 рядків × 26 стовпців** на аркуш – аналогічна Microsoft Excel та OpenOffice Calc.

### 5.1 Базове Керування

| Дія | Опис |
|-----|------|
| Клацання на клітинку | Вибір клітинки |
| Подвійне клацання / F2 | Редагування клітинки |
| `Enter` | Підтвердити, перейти до наступного рядка |
| `Tab` | Підтвердити, перейти до наступного стовпця |
| `Escape` | Скасувати редагування |
| `Ctrl + Z` | Скасувати |
| `Ctrl + Y` | Повторити |
| `Ctrl + C` | Копіювати |
| `Ctrl + V` | Вставити |
| `Ctrl + X` | Вирізати |
| `Ctrl + B` | Жирний |
| `Ctrl + I` | Курсив |
| `Ctrl + U` | Підкреслений |
| `Ctrl + F` | Знайти та Замінити |
| `Del` | Очистити вміст клітинки |

**Множинне виділення:** Перетягніть мишею або `Shift + Клацання`.

**Маркер заповнення:** Перетягніть синій квадрат у нижньому правому куті клітинки, щоб скопіювати значення або формули.

### 5.2 Рядок Формул

```
┌─────────┬────┬───────────────────────────────────────────┐
│   A1    │ fx │  =СУММ(A1:A10)                            │
└─────────┴────┴───────────────────────────────────────────┘
  ↑          ↑              ↑
  Адреса кл.  Символ          Поле вводу (відображає необроблену формулу)
```

- **Адреса клітинки**: Показує активну клітинку або виділений діапазон (напр. `A1:B5`)
- **Символ fx**: Вказує на введення формули
- **Поле вводу**: Завжди відображає необроблену формулу – не результат

**Відображення помилок:** При помилках формули на кшталт `#ПОМИЛКА`, `#ПОСИЛ!`, `#ДІЛ/0!` тощо поле обводиться червоною рамкою, а помилка відображається праворуч.

### 5.3 Введення Формул

Введення формули завжди починається з `=`:

```
=СУММ(A1:A10)              Сума
=СРЗНАЧ(B1:B5)             Середнє
=МАКС(C1:C100)             Максимум
=МИН(D1:D50)               Мінімум
=СЧЁТ(A:A)                 Підрахунок
=ЕСЛИ(A1>0;"Так";"Ні")     Умова
=ВПР(...)                  Вертикальний пошук
=ОКРУГЛ(A1; 2)             Округлення
```

#### Посилання у Формулах через Клацання Мишею

1. Введіть `=` або функцію на кшталт `=СУММ(`
2. Клацніть на потрібну клітинку → Адреса буде вставлена
3. Перетягніть для діапазонів → Діапазон буде вставлений (напр. `A1:B10`)

### 5.4 Формули між Аркушами

Посилання на інші аркуші – як в Excel:

```
=Аркуш2!A1                     Окрема клітинка з Аркуш2
=СУММ(Аркуш2!A1:B10)           Сума з Аркуш2
=Аркуш1!A1 + Аркуш2!B5         Комбінація кількох аркушів
=СРЗНАЧ(Продажі!C1:C100)       Діапазон з аркуша "Продажі"
```

**Через клацання мишею:**
1. Введіть `=` у клітинку
2. Клацніть на вкладку іншого аркуша → Вставляє `Аркуш2!`
3. Клацніть на потрібну клітинку → Додає `A1`

### 5.5 Числові Формати

Через символ **123** на панелі інструментів:

| Формат | Приклад | Опис |
|--------|---------|------|
| Стандартний | 1234.5 | Без форматування |
| Число | 1 234,56 | Формат з 2 десятковими знаками |
| Валюта | 1 234,56 ₴ | Формат валюти |
| Відсоток | 12,5% | Значення × 100 зі знаком % |
| Тисячі | 1 234 | Цілі числа з роздільником тисяч |
| Дата | 24.02.2026 | Формат дати |

### 5.6 Об'єднання Клітинок

1. Виберіть діапазон клітинок
2. **Вставка → Об'єднати клітинки**
3. Зберігається лише значення верхньої лівої клітинки

**Розділення:** **Вставка → Розділити клітинки**

### 5.7 Умовне Форматування

1. Виберіть діапазон
2. **Редагувати → Умовне форматування...**
3. Оберіть правило: Більше ніж / Менше ніж / Дорівнює / Між / Текст містить
4. Оберіть кольори тексту та фону
5. **Додати**

**Видалити:** Виберіть діапазон → **Редагувати → Видалити умовне форматування**

### 5.8 Закріпити Рядки/Стовпці

1. Виберіть клітинку, з якої починається прокрутка
2. **Перегляд → Закріпити рядки/стовпці**
3. Всі рядки вище та стовпці ліворуч будуть закріплені

**Зняти закріплення:** **Перегляд → Зняти закріплення**

### 5.9 Коментарі

1. Виберіть клітинку → **Вставка → Додати коментар** → Введіть текст → OK
2. Клітинки з коментарями показують **червоний трикутник** у верхньому правому куті

### 5.10 Знайти та Замінити

`Ctrl + F` або **Редагувати → Знайти та Замінити**:
- **Знайти**: Навігація ◀ ▶
- **Замінити**: По одному або всі одразу

### 5.11 Вставка/Видалення Рядків та Стовпців

Через меню **Вставка**: Вставити рядок вище/нижче, стовпець ліворуч/праворуч, видалити рядок/стовпець.

### 5.12 Рядок Стану

Внизу автоматично відображається статистика для множинного виділення:

```
Σ Сума: 12 345   ⌀ Сер.: 1 234   ↓ Мін: 100   ↑ Макс: 5 000   Кількість: 10
```

### 5.13 Імпорт / Експорт

| Функція | Формат | Опис |
|---------|--------|------|
| Імпорт | `.xlsx`, `.xlsm` | Відкрити файли Excel |

## 6. SEPShow – Презентації

SEPShow — редактор презентацій на основі слайдів, побудований на **Konva** (рендерер полотна).

### 6.1 Інтерфейс

| Область | Опис |
|---------|------|
| **Ліва бічна панель** | Список слайдів – попередній перегляд, зміна порядку перетягуванням |
| **Головна область** | Редактор полотна активного слайду |
| **Панель інструментів** | Інструменти, експорт, режим презентації |
| **Нотатки доповідача** | Нотатки до поточного слайду |

### 6.2 Режим Презентації

Натисніть **Презентувати** для повноекранної презентації:
- Навігація: клавіші зі стрілками `→` / `←` або клацання
- `Esc` завершує режим презентації
