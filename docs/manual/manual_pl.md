# Podręcznik użytkownika SEPOffice

Witamy w **SEPOffice**, inteligentnym pakiecie biurowym napędzanym przez AI. SEPOffice łączy klasyczne narzędzia biurowe z najnowocześniejszą lokalną sztuczną inteligencją, aby pomóc Ci pracować szybciej i inteligentniej.

![Przegląd](../assets/overview.png)

---

## 2. SEPWrite — inteligentny edytor
- **Autouzupełnianie AI**: Zacznij pisać i pozwól, aby Eliot zasugerował kolejne słowa.
- **Optymalizacja tonu**: Zaznacz tekst i poproś Eliota, aby nadał mu bardziej profesjonalny lub zwięzły charakter.

![SEPWrite](../assets/sepwrite.png)

---

## 3. SEPGrid — dynamiczne arkusze kalkulacyjne
- **Generowanie formuł AI**: Po prostu opisz, czego potrzebujesz, a Eliot napisze formułę.
- **Wizualizacje**: Błyskawicznie twórz wykresy i tabele.

![SEPGrid](../assets/sepgrid.png)

---

## 4. SEPShow — efektowne prezentacje
- **Magic Layout**: Jedno kliknięcie, aby idealnie rozmieścić elementy.
- **Vibe Check**: Pozwól Eliotowi sprawdzić spójność wizualną Twoich slajdów.

![SEPShow](../assets/sepshow.png)
