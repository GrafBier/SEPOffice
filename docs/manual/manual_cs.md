# Uživatelská příručka SEPOffice

Vítejte v **SEPOffice**, vašem inteligentním kancelářském balíku poháněném umělou inteligencí. SEPOffice kombinuje klasické nástroje produktivity s nejmodernější lokální AI, aby vám pomohl pracovat rychleji a chytřeji.

![Přehled](../assets/overview.png)

## 2. SEPWrite – Inteligentní editor
- **Doplňování pomocí AI**: Začněte psát a nechte Eliota navrhovat další slova.
- **Optimalizace tónu**: Zvýrazněte text a požádejte Eliota, aby jej učinil profesionálnějším nebo stručnějším.

![SEPWrite](../assets/sepwrite.png)

---

## 3. SEPGrid – Dynamické tabulky
- **Generování vzorců pomocí AI**: Stačí popsat, co chcete, a Eliot napíše vzorec.
- **Vizualizace**: Okamžitě vytvářejte grafy a tabulky.

![SEPGrid](../assets/sepgrid.png)

---

## 4. SEPShow – Působivé prezentace
- **Magické rozvržení**: Jedno kliknutí pro dokonalé uspořádání prvků.
- **Vibe Check**: Nechte Eliota zkontrolovat vizuální konzistenci vašich snímků.

![SEPShow](../assets/sepshow.png)
